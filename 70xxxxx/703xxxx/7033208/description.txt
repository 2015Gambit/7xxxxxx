This is an old PC game from 2003. It is compatible with Windows XP or older systems ONLY! I did NOT get this to work on Windows Vista or Windows 7, even with compatibility settings.

1. unpack archive
2. Mount the image to virtual drive, or burn image to disk
3. Run the installer
4. Install Direct X, if necessary
5. Install the Game and wait for the installation to complete
6. Play the game

Note: You must have the image mounted or disk inserted to play the game


***This is now an ISO so it can be mounted with daemon tools***
