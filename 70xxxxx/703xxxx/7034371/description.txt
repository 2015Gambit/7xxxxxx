


description:

Changes the desktop wallpaper on regular time intervals

Desktop Wallpaper AutoChange is a handy and reliable utility designed to rotate desktop wallpapers.

When you run it first, you should set the Image Path and Time-span then click [Save Setting] button, then click [Run] button.

This application will be minimized, and the process begin. When tick , the Desktop Wallpaper will be replaced by a random image. If you want to show the application window, you should mouse click the Tray icon on the task bar, then the application window will should and the wallpaper change process will be stop.

You can let this application run with windows systemby check [Auto-Run when Windows Start]. Under windows 7, you should run this application with Administrator accout, then do this setting.
