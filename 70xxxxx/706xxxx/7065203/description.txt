Nintendo DS Emulator with games
Running successfully on pc and Nintendo DS console.

For Pc
 1) Run the application desmume.exe
 2) After that in file option select open rom and then select the game from the rom   folder in the folder NINTENDO_DS_EMULATOR.
 3) Done
 
For Nintendo DS console.
If you need to run these games in Nintendo DS console then just copy the games from the rom folder into your DS memory stick and enjoy the games on Nintendo DS console.
