                               NEW IN MINECRAFT 1.2 
-----------------------------------------------------------------------------------


1. Ladders regained their collision box, and vines are climbable when they have a                              
    solid block behind them

2. Added jungle biome (requires a new world) with new tree and leaf types

3. Updated language files

4. Dispensers will place mobs instead of releasing the spawner egg

5. Zombies have new path-finding AI

6. Wolves have new AI and can be bred (when tame)

7. New jungle-specific mob that can be tamed

8. Skeletons have new AI and a few new behaviors

7. Various minor bug fixes and changes

9. Added an experience item to creative mode

10. Added a flame itemâ€¦

11. Many mob-related bug fixes (and some new behaviors)

12. Ocelots are easier to tame, and cats can be told to sit

13. Many small bug fixes and tweaks

14. Villagers will try to detect village houses and live in them

15. Many minor bug fixes and tweaks

16. Doors have been changed to properly detect if they are open or closed. Double         
    doors in existing maps will need to be updated

17. Villagers will move indoors during night and rain, and socialize with each other

18. Zombies will break down wooden doors on hard difficulty

19. Added cat sounds

20. Better movement for mobs in water

21. Added rare mob item drops

22. Maximum build height has been increased to 256 (was 128)

23. Empty sections of the world are not loaded into memory

24. Block ids have been increased to 4096 (was 256) by adding a 4 bit data layer    
   (similar to how meta data is stored)

25. Block ordering have been changed from x/z/y to y/z/x in order to improve  
    compression

26. Packets for sending chunks have been updated (a full 128-high chunk is smaller  
    than the old format, and a chunk with lots of empty space is much smaller)

27. Biomes are saved per x/z column, which means they can be altered by tools

28. Multi-player light calculations do no longer cause affected blocks to be 
    transmitted to the clients, instead the clients will recalculate the light on   
    their own

29. Villagers will repopulate villages based on how many houses there are available

30. Some nights in villages will be worse than othersâ€¦

31. Added a redstone-controlled light source

32. Decreased chance for the rare mob drops

33. All animals use the new AI system now

34. Added a defense mob to villages

35. Added up-side-down stairs

36. Many changes to the world and lighting system

4. now just change the current .jar file that you downloaded to minecraft.jar

MAC - 

1. goto Finder/User/Library/Application Support/minecraft
