iSleep is a simple timer for the Mac, allowing you to set when your Mac will restart, shutdown, and sleep.

What's New:

- Improved support for OS X Tiger.

- Improved support for OS X Leopard.

- Improved support for OS X Snow Leopard.

- Improved support for OS X Lion.

- Bug fixes.

- Slight icon change.

System Requirements:

Intel Mac, OS X.4.3 and above (Lion compatible)
