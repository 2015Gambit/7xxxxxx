==================================
 Perfect 2012a chess opening book
==================================

Size: 4.07 MB (packed), 29.5 MB (unpacked)

-released January 20, 2012
-an excellent short chess openings book!
-limited to the first 8 moves in a chess game
-designed especially for engine vs engine matches
-an interesting and useful addition to any chess library!

Works with most chess GUIs:
Aquarium, Arena, Chessbase/Fritz/Hiarcs/etc, ChessGUI, ChessMaster, ChessPartner, Shredder, Winboard

Installation:
-unpack the .rar file
-read the ReadMe file
-follow the instructions
-access with your favorite GUI
