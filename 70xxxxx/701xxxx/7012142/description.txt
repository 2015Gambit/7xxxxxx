Description
A minimal Debian based Linux system with fully functional bash shell (with GNU coreutils, not BusyBox), TCP/IP networking with DHCP client and APT setup to be able to install any package from the Debian repositories.  
http://sourceforge.net/projects/bachatalinux/

*.iso files are CD-ROM images with a bootable self-contained installer. Burn on a CD-R disk and boot from it to install on a physical machine, or use directly to install in a virtual machine. *.img files are disk images with a bootable self-contained installer. Write to an USB disk and boot from it to install on a physical machine. Use '486' if you have an i486 or early Intel Pentium CPU. Use '686' if you have an modern Intel or AMD CPU. Use '686-bigmem' if you have an modern Intel or AMD CPU with PAE support and more than 4 GB RAM.  
http://sourceforge.net/projects/bachatalinux/files/
