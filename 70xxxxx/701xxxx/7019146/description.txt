Full Game Collections: Pokemon Mini
===================================

This torrent is intended for sharing the full collection of Pokemon Mini roms,
compiled to match the No-Intro[1] database, dated 2008-11-30.

Contents
--------

  sets/                  All PokeMini roms, torrentzip'ed
  data/                  Files used to build this collection and other info
    PokeMini_have.txt    List of games in the collection
    *.dat                Corresponding No-Intro datfile

Tools used
----------

  TorrentZip 0.2 [2]
  ClrMamePro 4.02 [3]

ClrMamePro and TorrentZip Guide: [4]
