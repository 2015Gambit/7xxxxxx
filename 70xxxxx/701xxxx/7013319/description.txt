Set-Up Server (Unix):
Install netatalk on Linux box. Unarchive Volume68k.tar.gz to the AppleShare Volume.

Set-Up Client (Classic Mac):
Install System 7.5.3 and Update to 7.5.5.
Then, install Open Transport 1.1.2, and AppleShare Client 3.7.4.

After, you'll be able to connect your server!! Enjoy apps and games.
