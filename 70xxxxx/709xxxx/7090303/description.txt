Full Game Collections: Neo Geo Pocket & Neo Geo Pocket Color
============================================================

This torrent is intended for sharing the full collection of SNK NGP and NGPC
roms, compiled to match the No-Intro[1] databases, as follows:

SNK - Neo Geo Pocket         2012-02-28
SNK - Neo Geo Pocket Color   2012-02-27

Contents
--------

  ngp/                   Neo Geo Pocket ROM sets
    sets/                All NGP roms, torrentzip'ed
    data/                Files used to build this collection and other info
      NGP_have.txt       List of games
      *.dat              Corresponding No-Intro datfile

    sets/                All NGPC roms, torrentzip'ed
