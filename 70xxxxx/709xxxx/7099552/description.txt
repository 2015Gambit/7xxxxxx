Description
Kebox is a Debian-based linux distribution. It use Openbox and GTK+ applications. Currently it is offered in French version.
Kebox est une distribution linux basée sur Debian. Elle utilise Openbox et des applications GTK+. Actuellement elle est offerte en version française. 
http://sourceforge.net/projects/keboxlinux/

README 
Kebox k[ ] use Debian testing branch. live-build scripts from Debian Live Project was used to build Kebox Live system image. Here are some examples of boot parameters: locales=fr_CA.UTF-8 keyboard-layouts=ca keyboard-variant=fr timezone=America/Montreal xorg-resolution=1024x768 nox11autologin todo make a bilingual system (english and french) include the debian installer make a 64 bits version make a web site on kebox.org ...
