Full Game Collections: Game Gear
================================

This torrent is intended for sharing the full collection of Sega Game Gear
roms, compiled to match the No-Intro[1] databases, as follows:

Sega - Game Gear         2011-11-29

Contents
--------

    sets/                All GG roms, torrentzip'ed
    data/                Files used to build this collection and other info
      GG_have.txt        List of games
      *.dat              Corresponding No-Intro datfile

Tools used
----------

  TorrentZip 0.2 [2]
  ClrMamePro 4.02 [3]

ClrMamePro and TorrentZip Guide: search the web for "Torrentzip for Dummies".
If the host you find is down, try [4].
