Para los que estéis acostumbrados a WifiWay y no queráis tener que hacer uso del Live CD cada vez que queráis usar este magnífico programa, aquí os presento la versión 3.4 ya instalada en una máquina virtual y con las "Guest Additions" igualmente instaladas, de manera que sólo tenéis que, tras instalar VirtualBox, bajar el archivo, descomprimirlo, crear una máquina virtual “Other Linux Kernel 2.6″ en VirtualBox y usar como disco duro el fichero “Wifiway34.vdi” que habéis obtenido con la descompresión.

Además, también se incluye un tutorial con el que realizar futuras instalaciones de WifiWay en una máquina virtual.
