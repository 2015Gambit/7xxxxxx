Supported Operating System:-

Windows 2000
Windows XP
Windows server 2003 


How to install steps

1:- Before running Wilcom setup disable your antivirus software and their scanning services because some antivirus softwares treat Wilcom files as they contain viruses which is actually false positive and there are no viruses in their files.

2:- After installation of Wilcom provide their path in exclusion lists of your antivirus software specially Bin folder Which you can found in ESWIN folder, so antivirus software not able to delete their essential files which are required to run wilcom.

3:- After followed above steps now you can enable your antivirus software and their scanning services which you stopped before running Wilcom setup.
