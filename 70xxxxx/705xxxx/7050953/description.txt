1) Mount iso.


2) Install game.


3) Copy all the content of the folder \"INDRIT\" to

   \"C:\\Program Files (x86)\\Microsoft Games\\Gears of War\\Binaries\"

   except \"startup.exe\"


4) Find in \"C:\\Program Files (x86)\\Microsoft Games\\Gears of War\\Binaries\"

   the file named \"GearsTU\" and install it


5) Find in \"C:\\Program Files (x86)\\Microsoft Games\\Gears of War\\Binaries\"

   the file named \"GearsPC_TU3_Microsoft\" and install it.


6) Now copy and replace the file \"startup.exe\" from the folder \"INDRIT\" to

   \"C:\\Program Files (x86)\\Microsoft Games\\Gears of War\\Binaries\"


7) Create a shortcut to the desktop , have fun.


NOTE : In the folder named \"INDRIT\" you will find an ICON too , just to replace 

       the ugly startup\'s
