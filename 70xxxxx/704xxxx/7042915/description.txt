This is the fill client download file for the Silkroad Salvation private SRO server.  To play, follow these instructions:  

1.  Download this file and place it in your program files.
2.  Create a shortcut to \"silkroad.exe\" and place it on your desktop
3.  Make a game account at http://silkroadsalvation.com/index.php?pageid=reg
4.  Launch and sign in!

Silkroad Salvation is a classic bot-free silkroad online server that caters to english speaking players. The server has an oldschool 70 cap with redesigned 110 content to fit the new cap. Active GMs, custom content, and weekly updates unlike any other server out there.
