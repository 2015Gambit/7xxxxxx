 QUAKE LIVE Premium maps.

 Place ALL of them in the installed directory:
 |
 |--Windows XP users:
 | |
 | |--%appdata%id softwarequakelivebaseq3
 |
 |
 |--Windows 7 users:
   |
   |--<username>AppDataLocalLowid Softwarequakelivebaseq3

 Then, if you already have Quake Live website running, you must refresh it in order to recognise all the maps available so the game doesn't show you an error.
