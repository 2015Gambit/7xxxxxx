Sophocles 2007 Beta with Reg fix.

Instructions:

1. Install Sophocles 2007.

2. Once installed, make sure the program is not open.  Double click the file soph2007reg.reg

3. After the confirmation that the reg file was added open up Sophocles.  You should not be able to print and export.  If not, close and repeat step 2.  Sometimes, it doesn't take on the first go round.

Enjoy :)
