Photoshop Plugin - Grain Surgery V 2 - Portable  :
-------------------------------------------------

This is the latest (and last) version 2 of Photoshop Plugin - Grain Surgery V 2 . 

This Plugin has  3 Filter :

1. Match Grain

2. Create Grain

3. Remove Grain

Each of these Filters have their own controls, using which you have infinite 

control on grain in your photos.

The controls are simple and intiutive. Grain control can make your photos look 

old, brand new, rusty etc. etc.

This Plugin is Compatible with Photoshop V 7 to Photoshop  V CS 5.1  and 

Windows  XP  to Win 7.

Installation : 

1) Unrar

2) Copy / Paste the Plugins in the Plugin Folder typically located at - 

C:Program Files - Adobe - Adobe Photoshop CS5.1 - Plug-ins (in Win 7)

In Photoshop the Plugin is available at Filters - GraiSurgery - (Filters).
