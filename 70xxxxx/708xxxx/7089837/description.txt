Free Amateur Radio Software for Windows

Ham Radio Deluxe (HRD) Version 5.0 is a suite of free Windows programs providing computer control for commonly used transceivers and receivers. HRD also includes mapping, satellite tracking and the digital mode program Digital Master 780 (DM780).

HRD is designed for Windows XP, Vista and Windows 7), also Internet Explorer 6.0 (or higher) is required. It may work with Windows 98 but this is not supported. The policy is to support Windows versions which are supported by Microsoft.

Other operating systems are not supported at this time although there is discussion on the forums. HRD runs well under Virtual Machines of which there are at least two available freely.
