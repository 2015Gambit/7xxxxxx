KEYGEN - and how to use :
------------------------

Please note that the Keygen will be uploaded with each torrent. The Keygen is 

common to all the Plugins / Programs and can generate ### for each of them.

No need to start a plugin after installation & register. 

1) Just open Keygen 

2) Select an Akvis Product from Drop down Menu & hit "Make Key File". Let all 

other Parameters, like "Install Directory of selected products "  &   "Public 

shared folder (License file MUST be saved here )" remain as default .

3) The Plugins All get Registered automatically. They can be checked from 

Standalone as well as from plugins from within Photoshop. They are all get 

registered after the Keygen "Make Key File" button is clicked ! (ie. no need to 

copy or paste or replace or patch any thing).

4) Each Plugin / Program installs Shortcut on the DeskTop, which can be used to 

start the Standalone Program.
