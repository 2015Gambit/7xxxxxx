****************************************************************
     1. System Requirements
****************************************************************

Minimum:
* Windows XP/Vista/7
* Intel Core 2 Duo-Prozessor
* 1 GB RAM (XP)/2 GB RAM (Vista)
* GeForce 8800 GTX

Recommended:
* Windows XP/Vista/7
* Intel Quad Core
* 2 GB RAM
* GeForce GTX 295


Please make sure to have the latest drivers for your system installed.
ArcaniA - Gothic 4 requires NVIDIA PhysX, VCRedist and DirectX which are all part of the installation package.

     2. General information
****************************************************************

ArcaniA - Gothic 4 saves config files and savegames under "My DocumentsArcaniA - Gothic 4".
Manual changes to config files can cause problems!

ArcaniA - Gothic 4 is using the securom copy protection system featuring online activation.
If you want to perform the activation without an active internet connection, need general support
with the copy protection system or you want to revoke your activation please check out this
ArcaniA copy protection support page: https://support.securom.com/ArcaniAunlock/
