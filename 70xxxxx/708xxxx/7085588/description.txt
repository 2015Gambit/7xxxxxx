Linux distribution with high anonymity for reporters and activists 
Description

Linux distribution based-on Debian GNU/Linux 6.0.3. The system includes all sorts of open source tools for gaining anonymity on the internet. It also provides secured communication and collaboration tools and the documentation to start using the included software.
