Arduloko_OS and an operating system based on Ubuntu 11.04 64-Bit (natty) directed especially to students and professionals of electronics. 
The system is distributed with several programs for arduino and other equipment.

Arduloko_OS é um sistema operacional com base no Ubuntu 11.04 64-bits (natty) dirigido especialmente aos estudantes e profissionais de eletrônica. O sistema é distribuído com vários programas para arduino e outros equipamentos.
