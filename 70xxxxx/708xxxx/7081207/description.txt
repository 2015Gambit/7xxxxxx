These are tutorials for Pixara Twisted Brush. These tutorials are in .swf videos. 

You can download the Free Irfan View program to view these tutorials.

Pixara Twisted Brushes is an excellent program to retouch and paint your photos and 

images. It provides unlimited brush sizes / thickness in various hues and colors. 

the results can be saved in innumerable extensions and imported / opened in 

Photoshop / Illustrator / CorelDRAW etc.

What you would require to fully exploit this little known but incredible program :

1) Pixara Twisted Brush program.

2) Irfan View (A Free Program), or any other program which can play .swf video 

files for the tutorials.

Once you are addicted to "TwistedBrushes" , you will forget all about using 

photoshop for brushing and painting needs.
