assassin's creed brotherhood 
requisitos 


Configuración mínima
Sistema operativo: Mac OS X 10.5/Mac OS X 10.7
Procesador: Intel Core 2 Duo de 3,06GHz
RAM: 3GB 
Tarjeta de vídeo: Tarjeta dedicada con 256MB de memoria y Shader Model 3.0 o superior 
                         (ver lista de tarjetas soportadas)
DVD-ROM: Lector DVD-ROM de doble capa
Espacio libre en disco duro: 9GB
Periféricos soportados: Teclado, ratón

* Este producto no soporta sistemas operativos anteriores a Mac OS X 10.5
* Este producto no soporta Macs basados en la CPU PowerPC


Configuración recomendada
Procesador: Core i5 2,6GHz o superior 
RAM: 4 GB o más
Tarjeta de vídeo: Tarjeta dedicada ATI Radeon® HD 4850, NVIDIA GeForce® 200 o superior, 
con 512MB de memoria gráfica
 

Tarjetas de vídeo soportadas en el momento del lanzamiento:
ATI® RADEON® series HD 4000/5000
NVIDIA GeForce® series 100/200
