Ultra Keyboard is an advanced touch-screen keyboard, with many features geared towards giving you a better Android keyboard experience.

What's in this version : (Updated : Aug 16, 2012)

    Fixes:
        Missing text cursor in full-screen editor.
        Speech-to-text auto-space behavior (and added a setting to disable it).
        Soft keyboard popping up when using a physical keyboard.

Required Android O/S : 1.6+
