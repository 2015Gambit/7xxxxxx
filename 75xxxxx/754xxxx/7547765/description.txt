ProgDVB Professional 6.86.4 (Multi/PL) + Aktywator

Systemy:Windows (x86/x64 bit)

ProgDVB Professional to program, dzięki któremu możemy posłuchać radia i obejrzeć telewizję cyfrową. Aplikacja pozwala na odtwarzanie stacji nadających przez internet, a także na tych odbieranych drogą satelitarną (przy użyciu odpowiedniech kart rozszerzeń).

ProgDVB obsługuje standardy DVB-S (satellite), DVB-S2, DVB-C (cable), DVB-T, IPTV, HDTV. Program udostępnia m.in.: EPG, napisy, teletekst, nagrywanie z radia i TV, harmonogram zadań, obsługę wielu wtyczek oraz kilka innych funkcji. Możliwe jest również oglądanie niektórych kanałów kodowanych, lecz potrzebne do tego są wtyczki.

Wersja Professional w porównaniu z wersja Standard pozwala na nagrywanie tv/radia internetowych, obsługuje DVB-S2 i HDTV, kilka kanałów audio per program i funkcję PIP.

Uwaga: program wymaga zainstalowanego pakietu Microsoft .NET Framework 2 lub nowszego.

ENG

ProgDVB - software which allows you to watch SAT-Television and listen to Radio channels directly from satellite by using DVB-PCI cards with hardware decoders on the board, SAT-dish, and personal x86-compatible computers.
ProgDVB is a very useful application for those who want to watch television shows and programs on their computer.
ProgDVB has options for working with network broadcasting and Audio/Video recording of the stream to different digital-media formats. ProgDVBR is in development, so new versions will be available for download from time to time.

Requirements: ProgDVB 6x and 5x requires Microsoft .NET Framework Version 2.0 or higher.
