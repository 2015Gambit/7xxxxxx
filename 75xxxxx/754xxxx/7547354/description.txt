Requirements: Android
Overview: Download files from Usenet newsgroups directly to your phone!
Image


Requires a valid Usenet account and access to NZB index files. Handles any DIRECT links in browser to NZB files. Integrates with Newzbin Search application for easy loading of NZB files.

Changelog:
* User Interface Improvements:
- Smaller font
- Filesize displayed for each NZB entry
- List of all queued downloads on the status tab
- Refresh Media Library button
- Refresh NZB list button
