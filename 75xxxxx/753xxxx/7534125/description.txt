Windows XP Service Pack 3 Español con IE8, WMP11 y WLE2009
----------------------------------------------------------

Actualizado a Agosto de 2012
Basado en la ISO oficial MSDN VL:

Archivo: es_windows_xp_professional_with_service_pack_3_x86_cd_vl_x14-74009.iso
MD5: 7149a9dd2f13e00ce54df01ff7a69a59

Esta compilación está pensada para ofrecer una instalación desatendida de Windows XP SP3 edición Española lo más limpia y actualizada posible, incluyendo:

- Todas las actualizaciones hasta la fecha.
- Internet Explorer 8 (sustituye a la versión 6).
- Windows Media Player 11 (sustituye a la versión 9).
- Windows Live Mail 2009 (sustituye a Outlook Express).
- Windows Live Messenger 2009 (sustituye a Messenger).
- Pack de 53 Temas para XP.

No se incluye ningún software extra de terceros, siendo el usuario libre de utilizar el que mejor cubra sus necesidades.

Se incluye una carpeta de Archivos para que el usuario pueda asignar/modificar su propia clave de Windows XP, entre otras cosas.
