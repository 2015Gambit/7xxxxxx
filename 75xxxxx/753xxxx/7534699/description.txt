Borderlands is a space western first-person shooter with role-playing elements, that was developed by Gearbox Software for PlayStation 3, Xbox 360, Microsoft Windows and Mac OS X. It was first revealed in the September 2007 issue of Game Informer magazine.[10] The console versions of the game were released in North America on October 20, 2009, and were released in PAL countries on October 23. The console version release for the Japanese market was made available on February 10, 2010. The Windows version was released on October 26 for North America and then on October 29 internationally.[5] The Mac OS X version of the game was released on December 3, 2010 by Feral Interactive.[11] A sequel, Borderlands 2, will be released on September 18, 2012
