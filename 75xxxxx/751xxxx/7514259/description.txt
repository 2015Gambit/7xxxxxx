

F-Secure Client Security to rozwiązanie zapewniające ochronę korporacyjnych stacji roboczych z systemem Windows i Linux. Produkt oferuje automatyczną ochronę w czasie rzeczywistym przed zagrożeniami z internetu (m.in. wirusami, oprogramowaniem szpiegującym czy programami typu rootkit). W ramach zakupionej licencji istnieje możliwość zarządzania programem z poziomu intuicyjnej konsoli zdalnej administracji F-Secure Policy Manager.
 
F-Secure Client Security jest kompletnym pakietem zabezpieczającym dla komputerów stacjonarnych i notebooków z systemami Windows oraz Linux. Produkt oferuje bezkonfliktową integrację z różnymi technologiami ochrony komputera. Należą do nich kwerendy serwera do sieci ochrony w czasie rzeczywistym, blokowanie behawioralne czy zaawansowana heurystyka. Uzupełnieniem ochrony w F-Secure Client Security są oczywiście sygnatury wirusów udostępniane przez producenta. Dzięki temu możliwe jest automatyczne rozwiązywanie problemów ze złośliwym oprogramowaniem. Nową funkcjonalnością jest ochrona użytkownika przed szkodliwymi witrynami sieci Web, które są blokowane przez oprogramowanie. Nowa wersja F-Secure Client Security 9 oferuje przebudowany interfejs użytkownika, który jest znacznie prostszy i łatwiejszy w użyciu.


Systemy operacyjne
 
- Microsoft Windows XP z dodatkiem SP3 (32-bitowe tylko) 
 - Microsoft Windows Vista (wszystkie 32-bitowe i 64-bitowe) 
 - Microsoft Windows Vista z dodatkiem SP1 lub nowszym (wszystkie 32-bitowe i 64-bitowe) 
 - Microsoft Windows 7 (wszystkie 32-bitowe i 64-bitowe) 
 - Microsoft Windows 7 z dodatkiem SP1 lub nowszym (wszystkie 32-bitowe i 64-bitowe)
