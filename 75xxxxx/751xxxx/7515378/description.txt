Use Phantom Drive on any computer to burn CDs, DVDs, Blu-ray discs and HD DVDs using any application program - with no physical media and no hardware disc recorder!Phantom Drive installs a virtual recorder in your system that can record all types of virtual CD, DVD and HD media. Create virtual CDs, DVDs, HD DVDs and Blu-ray discs from any application that provides a recording command, whether or not you have an optical recorder in the system.You can then insert the virtual media in the Phantom Drive to use it. Thanks to its support for the complete range of media types, its high recording speed, and its ease of use, Phantom Drive saves you time and money in many application scenarios.
Phantom Drive supports:
- All versions of Windows XP
- All versions of Windows Vista
- All versions of Windows 7
- Windows Server 2003 and 2008
