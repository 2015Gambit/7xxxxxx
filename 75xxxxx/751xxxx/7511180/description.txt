************************************ HOW TO USE ***********************************

1. Download them
2. Make am folder named 'mods' in the game or server directory
3. Extract the mods in the folder named 'mods'. Copy ONLY those you need it. More mods slows the server and gameplay.
4. Run mods by typing the following commands in console or placing it in server.cfg

  [ set fs_game mods/MODNAME ] or [ fs_game mods/MODNAME ]

like set fs_game mods/antinoob
