La tecnología HTML5 es la que tiene un presente y un futuro más claro en el mundo del desarrollo de contenidos para la web móvil y para los libros electrónicos para Tablets. Este curso video2brain te mostrará las técnicas más fundamentales para poder introducir vídeo, sonido y animaciones en tus contenidos HTML, y añadir interactividad Javascript enfocada a Tablets táctiles. De este modo podrás generar contenidos de vanguardia desde el primer día utilizando una programación simple, intuitiva y gratificante.
