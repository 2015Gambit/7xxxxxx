Requirements: Android 2.0+
Overview: Fun4Kids Kids Memory Game Plus
This memory app will keep your kids entertained, educated and quite in car, in a restaurant, on a plane or everywhere.

FEATURES
* 8 different kinds of memory game.
* Each game has different difficulty level.
* Colorful and vivid graphics.
* Develops recognition, memorization and concentration.
* Ladybug Memory Game - Level 1: These cute ladybugs will help your kids to recognize digits with colorful and vivid graphics.
* Duck Memory Game - Level 2 : This memory game will help your kids to recognize letters of alphabet with cute baby ducks and duck sound.
* Objects Memory Game - Level 3: Objects memory game has 33 most basic, must know objects with nice images and sounds.
* Transportation Memory Game - Level 3: Transportation memory game has 18 most basic, must known means of transportation with nice images.
* Fish Memory Game - Level 3: Fish memory game has different kinds of sea creatures. These 47 different sea creatures will keep your kid busy.
* Cute Dogs Memory Game - Level 3: Your kid will love these cute dogs with real barking sounds.
* Birds Memory Game - Level 3: This memory game includes different bird images with sound effects.
* Candy Memory Game - Level 4: This memory game has different kinds of candies, cupcakes, donuts and ice-creams. Your kid will enjoy the virtual Candy Shop. There are 50 different kinds of candies changes randomly for each game.
* Animals Memory Game - Level 5: Your kid will love these funny and cute animals with real animal sounds. It has 12 different backgrounds changing randomly.
35 diffirent animals with sounds.
