Description

Love match-3 games? Then you'll love Pop Farm!

Pop Farm features easy to learn, but challenging gameplay: tap groups of the same animal type to get points. The more animals in the group, the more points you get. Earn enough points to reach the next level and find new animals!


Game Highlights:
* challenge mode - see how many levels you can reach and how many new animals you can find
* OSX Snow Leopard, Lion and Mountain Lion support
* Fullscreen support - play PopFarm fullscreen on OSX Lion and Mountain Lion!
* unique soundtrack
* FAST - no long loading screens or lags; start playing the game immediately!

If you enjoy this game, please let us know what you'd like to see in the next update by writing a review. We hope you'll have as much fun playing Pop Farm as we did making it!

Requirements: OS X 10.6 or later, 64-bit processor
