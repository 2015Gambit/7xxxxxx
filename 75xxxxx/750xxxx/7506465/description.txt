Asrock Extreme Tuning Utility for Windows operating systems XP/Vista/7/XP-64/Vista-64/7-64, This software allows you to control the temperature of the motherboard, processor and fan speeds. Software also allows you to overclock the processor and recording a profile of overclocking (OC DNA). IES function is also included to reduce the number of phases used by the processor when it is a bit busy and reduces energy consumption.
