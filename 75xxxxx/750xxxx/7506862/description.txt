Description

Orbs Match is a fun and very addictive game for match-3 lovers!, playing it is very easy: just move the orbs to make matches of 3 or more before the orbs overflow or time runs out.

Features:
- Addictive gameplay
- Colorful graphics
- Great Music and SFX
- 50 challenging levels and more
- 3 Difficulty settings
- Close the game any time and resume later

What's New in Version 1.3

- Added Option for 1 Click Orb Flipping.
- Adjusted Orbs falling speed.
- Enhanced User Interface.
- Enhanced Font rendering.
- Updated Icon and Logo.
- Adjusted Music Tracks.
- Adjusted Mouse Size.
- Other Minor Changes.

Requirements: OS X 10.5 or later
