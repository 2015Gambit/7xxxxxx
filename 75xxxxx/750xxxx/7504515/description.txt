YouWave - an application for Windows, allows you to download and run Android applications without having to bother with downloading the Android SDK and Sun SDK. YouWave contains a virtual machine and Android installed from the distribution in two mouse clicks. After installing the program can run Android-application from your local drive or upload them via the Internet with free resources-sources Android-apps.

Key Features:
Support for Android 2.3 Gingerbread
Runs on Windows XP/Vista/7, 32/64 bit
Running Android applications and games on your PC
Download thousands of Android applications online
High performance - a quick way to run Android on your PC
Easy to use - easy to install
Easy to import and run the application
