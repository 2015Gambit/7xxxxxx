Installation:

1) Unzip the folder "The Elder Scrolls V Skyrim Update v1.7.7.0.6 Incl Dawnguard DLC."

2)Copy all files from the folder "The Elder Scrolls V Skyrim Update v1.7.7.0.6 Incl Dawnguard DLC" to your Skyrim installation main directory.

3) Create a desktop shortcut to SkyrimLauncher.exe

4)Start the Skyrim Launcher and at the Launcher menu go into Data Files and make sure that Danguard.esm is checked.

5) Play the game!


Note: The Dawnguard quest line should be given to you once you start the game. If it doesn't immediately start up then either go talk to a guard or go through a door and it should be given to you shortly there after.
