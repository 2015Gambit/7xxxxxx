SketchUp es un software diferente y divertido Google ha creado una herramienta 3D fácil e intuitiva que puede servir a todo tipo de profesionales: diseñadores, infógrafos, ilustradores, educadores… Aprende con este curso video2brain a usar este software con el que crear edificios, decoración de interiores, diseño de packaging, animaciones… cada profesional encontrará una utilidad a este estupendo software. Es la herramienta para crear los edificios que aparecen en Google Earth, pero la imaginación es el único límite.
