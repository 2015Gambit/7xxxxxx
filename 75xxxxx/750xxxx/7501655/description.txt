Installation instructions:

ESET NOD32
1. Open the main window of ESET NOD32
2. In the left pane, select the "Updates" -> "Activating"
3. Then select the "Activate by using the user name and password"
4. Enter user name (Username) and Password (Password) -> Activate
5. Then press the "Update virus signature database."

It's important! Turning off the system of early detection.
Open the main window of ESET NOD32 -> Settings -> Go to the advanced settings ... (F5) -> Unveiling the section "Tools" -> ESET Live Grid -> Remove the check mark from "Take part in the ESET Live Grid -> Click" OK ".

Dr.Web
1. On the taskbar in the system tray near the clock icon Find Doctor Web.
2. Click on the icon drweb right mouse button and the menu that appears, select "Tools" then "License Manager" and click the left mouse button.
3. A window "License Manager". In the window, see the information about the installed keys drweb.
4. In order to establish the key Dr. Web anti-virus must be added to the list of license manager. To do this, click on the search button and the menu, select the desired file in the selection window.

Avira

3. In the opened window looking for the words "I already have a valid license file HBEDV.KEY"
