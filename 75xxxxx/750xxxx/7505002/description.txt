"Summon Night: Swordcraft Story" is an action role-palying game by Banpresto for the Nintendo Game Boy Advance system. Atlus handled the English language translation of the game. It has a sequel, "Summon Night: Swordcraft Story 2".



"Summon Night: Swordcraft Story" é um jogo de RPG de ação de Banpresto para o sistema Nintendo Game Boy Advance. Atlus manejou a tradução do jogo para língua inglesa. Ele possui uma sequência, "Summon Night: Swordcraft Story 2".
