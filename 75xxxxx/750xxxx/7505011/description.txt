"Summon Night: Swordcraft Story 2" is an action role-palying game by Banpresto for the Nintendo Game Boy Advance system. It's the sequel of "Summon Night: Swordcraft Story". Atlus handled the English language translation of the game.



"Summon Night: Swordcraft Story 2" é um jogo de RPG de ação de Banpresto para o sistema Nintendo Game Boy Advance. É a sequência de "Summon Night: Swordcraft Story". Atlus manejou a tradução do jogo para língua inglesa.
