iChallenge Utilities 2.0:

Browser 2.0:
An updated version of Brawza, an internet explorer based web browser, now with tabs, that supports the most common Aero Features, such as  dragging to the top of the 
screen to maximise.

Picture Viewer 1.0:
This is a new utility, it is an extremely simple therefore small picture viewer, which supports a wide range of picture formats. It also supports the most common Aero Features, such as  dragging to the sides to compare.

Calculator 2.0:
This is an updated version of Calc and it is a simple calculator

The package also contains the developer files for people who would like to improve it, or make their own version.
