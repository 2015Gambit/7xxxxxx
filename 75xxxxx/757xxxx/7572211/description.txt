Visual FOX Pro v9 Service Pack 2

Is the complete package which can build and deploy database-type applications. Engine uses its own database to manage and process large volumes of information.

System Requirements:

* Procesor Pentium

* Microsoft Windows 2000 with Service Pack 3, Microsoft Windows XP, Microsoft Windows Server 2003

* 64 MB RAM minimum, 128 MB or more recommended
* 165 MB of available hard-disk space for installation, 20 MB additional hard disk space for Microsoft Visual FoxPro 9.0 Prerequisites
* CD-ROM or DVD-ROM drive
* Display Super VGA 800 X 600 or higher with 256 colors
* Mouse or other compatible device
