Calendar, GTD Tasks and Contacts - all in one! Pocket Informant is a fully integrated calendaring and GTD-based tasks solution for Android. We fuse together best-of-class calendaring with best-of-class tasks into one great solution. Even greater functionality has been added with the introduction of a Contacts view, which allows the user to fully access the Android contacts database from within Pocket Informant.

Version 2.0 is a major upgrade for Pocket Informant with many additional functions and features added.

Features:

    Fully integrated Calendar and Task Management
    Month, Day, Week, Column Week, Agenda, Task and Contacts Views
    Swipe between Months, Days, and Weeks
    Pre-made filters to quickly find selected Tasks
    Templates for Events and Tasks
    Ability to show/hide different Android calendars
    Optional Time-Bar and Mini-Text views for Calendar Month View
    Custom time/date selection controls for easy modification and navigation in main view
    Easily accessible commands for events or tasks directly from main views
    Highly configurable interface (with more than 100 different options!)

What's in this version : (Updated : Sep 3, 2012)

    Widget!
    Contacts View
    Multi-Select Mode in Task Views
    Multi-Select Mode in certain Calendar Views
    Pocket Informant Online Sync
    All Pocket Informant syncing is managed by the Android Sync Manager
    Multiple Accounts of the same sync type can be used at the same time.
    Now supports multiple internal Event and Task calendars
    ICS Support
    Ability to switch view modes on tablets between the Tablet View and Phone View.

Required Android O/S : 2.0+
