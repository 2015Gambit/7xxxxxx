IFBA TUNING:

To install cydia IFBA enter and search the latest version 0.5.
After we install it on the emulator: Settings / ROMS patches, select the first slot and we validate it, doing this creates a folder in / var / mobile / documents / IFBA / which is where volcaremos the ROMs.

PACKAGE CONTENTS:

Collecting all ROMs compatible emulator:

Capcom CPS1 CPS2 CPS3 (98 ROMs)
NeoGeo (127 ROMs)
PolygameMaster IGS (18 ROMs)


All playable funcionables and separated into folders
Total weight: 3.68 GB.


IMPORTANT:
To emulate NeoGeo and PGM need the BIOS that are included in the package and it will go on the same route with the ROMs.

PUESTA A PUNTO DE iFBA:

Para instalar iFBA entramos en cydia y buscamos la ultima versión 0.5.
Después de instalarla entramos en el emulador: Settings/ROMS patchs, seleccionamos el primer slot y lo validamos, haciendo esto nos crea una carpeta en /var/mobile/documents/iFBA/ que es donde volcaremos las ROMs.


CONTENIDO DEL PAQUETE:

Recopilación de todas las ROMs compatibles con el emulador de:


Capcom CPS1 CPS2 CPS3 (98 ROMs)
NeoGeo (127 ROMs)
PolygameMaster IGS (18 ROMs)


Todas funcionables y jugables separadas en carpetas
Peso total: 3,68GB.


IMPORTANTE:
Para poder emular NeoGeo y PGM necesitamos las BIOS que están incluidas en el paquete y que iran en la misma ruta junto a las ROMs.
