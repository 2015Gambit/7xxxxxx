How to use:

    Switch PESEdit.com EURO 2012 Patch OFF (skip if you don't have the EURO patch)
    Delete the folders 'pesedit', 'GDB' inside 'PES 2012/kitserver' (optional)
    Install the patch using install.exe, make sure to choose the directory you installed PES 2012 to.
    Start the game via pes2012.exe / start menu folder "PESEdit.com 2012 Patch" / PESEDIT SELECTOR
