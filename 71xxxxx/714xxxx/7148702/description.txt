Active@ Boot Disk (Win - DOS Edition) is a bootable CD/DVD/USB disk that allows you to boot up your computer and fix most startup and PC configuration problems. Active@ Boot Disk based on lightweight Windows VISTA (WinPE 2.1) operation environment and contains disk image, data recovery, partition management, password resetting, data erasure, network access tools and system utilities.
