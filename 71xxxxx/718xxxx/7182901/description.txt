Title: Rising Kingdoms
Genre: RTS / fantasy / isometric
Released: 17th June 2005
Developer: Haemimont Games
Publisher: Black Bean Games
Description:
Rising Kingdoms is a fantasy real-time strategy game, in which you build and expand your empire by capturing independent colonies and use them to aid you defeat your enemies. The game features three playable races - Humans, Foresters and Darklings - each with four special units called Champions, powerful leaders which can use different spells and abilities in combat. The campaign spans over 30 missions, 10 for each race.

Bonus content: artworks, avatars, soundtrack.

How to play:
1) Download the game;
2) Mount or burn disc image;
3) Run Install.exe to install (or press Install button if you have autorun enabled);
4) After installation is complete, run RKPatch12.exe to patch the game;
5) Play the game without the disc.
