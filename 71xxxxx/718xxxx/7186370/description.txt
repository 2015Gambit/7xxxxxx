                                  Release Info
                                  ************
This tool simplifies this process, and can convert from as small as one line to as many as thousands of lines of code in mere seconds.

HTML To PHP Converter has numerous useful features:

- Single file conversions
- Batch file conversions
- Copy and paste conversions
- Printing capabilities so you can look over your code on paper

HTML To PHP Converter can convert most HTML-based files: HTM, HTML, SHTML, and more to PHP with the click of a button.
