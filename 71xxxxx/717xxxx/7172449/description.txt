The main features of LibreOffice:

LibreOffice has some interesting features, partially or totally absent in the other office suites:
Cross-platform. Currently LibreOffice can be used on all popular platforms: Linux (32 - and 64-bit systems), Windows (2000 and later), MacOS, FreeBSD and Solaris.
Tools to help you create, edit and document management:
- To insert an image gallery;
- GPS to navigate the document and search;
- Stylist for creating, editing and management styles of document formatting;
- Data sources for importing data from databases or spreadsheets.
Supported file formats: MS Office. You can open and save documents in MS Office 97-2003. It also supports importing files from Office OpenXML (MS Office 2007).
Export to PDF. Export to the format supported by all components (except the base). If a particular extension can also import PDF to LibreOffice Draw.
Support for extensions. LibreOffice provides an opportunity to develop independent third-party extensions, providing additional functionality for the end user. Extensions can be easily installed and removed.
