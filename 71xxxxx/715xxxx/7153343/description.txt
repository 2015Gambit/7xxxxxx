Far Cry Delta Sector

* If you have a Missing MSVCP71.dll, please following the readme.txt in the installation folder and copy the MSVCP71.dll to your WindowsSystem32 folder.

Problem: In Vista/Windows 7, the mouse works in the menus but does not work in-game.

Fix: At the main menu, go to Options -> Control Options.
Select Default -> Yes.
Go back to the main menu and exit "Far Cry."
Restart "Far Cry."
