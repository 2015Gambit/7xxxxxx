Title: Scooby-Doo! Mystery of the Fun Park Phantom
Genre: adventure / top-down / puzzle-solving
Released: 1999
Developer: EAI Interactive
Publisher: SouthPeak Interactive
Description:
Once again Scooby-Doo and the gang are called to solve a case, this time the mystery of the Fun Park Phantom, who haunts the Gobs' family Fun Park, keeping visitors away and leaving the family with no choice but to close the park. You must find all the clues as you control the gang (except for Scooby himself, but if you find the Scooby Snacks that lay about the park, he will show you where to go to find that all important clue). The graphics and sound are true to the 70's TV show, with original cutscenes and hand drawn animation.

Bonus content: manual, soundtrack.

How to play:
1) Download the game;
2) Mount or burn disc image;
3) Run Install.exe to install;
4) Play the game with the disc inserted.

! Notice
Windows Vista/7 users: set the game's executable to Windows XP compatibility to run the game.
