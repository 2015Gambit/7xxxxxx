Phillips SAA7130 TV Card Setup

NOTE: I CANNOT guarantee that this program is OR is not compatible with Vista although it does work with ALL Versions of XP...

1. Go to the directory called "713xTVCard".

2. Run the program called "713xSetup.exe".

The setup program will install ALL of the programs necessary for your TV Capture card.

3. DO NOT RE-BOOT YOUR PC UNTIL THE WINDOWS DISAPPEARS.

4. Then you are safe to re-boot your PC.
