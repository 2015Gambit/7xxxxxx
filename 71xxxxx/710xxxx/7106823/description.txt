This is a web browser designed by me to be very fast at browsing and downloading from the web even on the slowest of internet connections. Also contains other applications built in like notepad, photo viewer, GIF image viewer, media player, swf player, basic calculator and a cookie viewer.

If you need any help, have ideas, improvements or if you find any errors or bugs, please email me at ben.steele33@gmail.com.

File is 26.8Mb in size.

This version is 3.5. On this version the secuirty has been updated and more features have been added. One of the updates is that you can force new searches into a new window automaticly.
This is one of the most powerful homemade web browsers you'll find on the web.
