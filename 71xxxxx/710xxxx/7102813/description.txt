Description
Minimalist is a project about building Ubuntu based distro with nothing but Gnome as DE and a few apps that would be usefull like synaptic, conky, nauitlus scripts etc. but nothing that will bloat it

Features
    * Based on Ubuntu Natty
    * Minimal Gnome 2 desktop env.
    * Great as a starting point after installation 
http://sourceforge.net/projects/minimalist/
