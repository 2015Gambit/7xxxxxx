Year: 2011
Version: 1.0.1.1
Developer: Visual Concepts
Language: English 



NBA 2K12 Minimum System Requirements

Intel Core 2 Duo 2.9 Ghz Processor
MAC OSX 10.7.x
1 GB RAM System Memory
256 MB vRAM dedicated ATI or nVidia graphics cards (minimum ATI x1300 or
NVIDIA 6600) Will be a hassle to run this game on Intel HD 3000 graphics cards even they run the game.



This is a Wineskin ported game and it works as long as you have the minimum system requirements for the game.

Before installing the game, change your OS X language to English if you use some language other than English.


Enjoy the game !!
