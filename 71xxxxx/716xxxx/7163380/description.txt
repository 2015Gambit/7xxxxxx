The A1 WMA Joiner 1.0 is a program for windows computers that allows you to Join, combine and merge many files of WMA (Windows Media Audio) and more audio files, such as MP3, OGG, SND, VOX, WAV, WMA to one audio file - with high quality results, easy to use and fast! Also, with this program you can also convert those format of audio file from one format to antohre.
