The Vector File (2002)

Nederlands
In het laboratorium van een satellietgrondstation op een tropische bergtop werkt de geleerde Gerry Anderson, terwijl hij op zijn dochter Mattie past.
Hij is zich niet bewust van het feit dat zijn wereld binnenkort op zijn kop zal worden gezet.
Als Mattie per ongeluk een print-out van een satellietbericht in haar tas stopt, heeft zij er geen flauw benul van,
dat deze de code voor de vernietiging van de mensheid bevat.

English
A geo-political thriller where Gerry Anderson finds that sometimes, everybody really is out to get you.
Gerry's daughter, Mattie, accidentally prints out a DNA code that in the wrong hands can end the human race.
The chase is on - the Russians, whose DNA code it is that Mattie has in her backpack,
and a group of terrorists compete to get the code from the person they all assume has it, Gerry.
With all this going on, Gerry's estranged wife Margaret is worried for her daughter's safety and finds herself another target in the race to find the code.
