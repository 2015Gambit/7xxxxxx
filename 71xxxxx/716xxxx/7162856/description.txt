Red Faction: Armageddon (Jimbus edition v1.0)
-----------------------


Features of this edition:
--------

  - Updated to 1.01 version and with the "Path to War" DLC  integrated.
  - It can be burnt to a single DVD-5.
  - It includes guides, trainers, the "TeknoRFA 1.0" (which enables L.A.N. multiplayer in English), the Skidrow files (which allows to play the campaign offline in a lot of languages) and the libraries the game requires.
  - The quality of some videos has been reduced.
  - No file is infected, though some antivirus say the opposite.
