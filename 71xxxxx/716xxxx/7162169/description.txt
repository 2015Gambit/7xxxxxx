Requires:

- PES 2012
- PESEdit Patch 3.1.1
- Tommsens Chantpack Baseline + BL AddOn



Installation:

- Copy folder "dt03.img" to "C:Program Files (x86)KONAMIPro Evolution Soccer 2012kitserverpeseditimg"
- Copy "Edit.bin" to "C:Users<your name>(Documents)KONAMIPro Evolution Soccer 2012save", replace existing



Info:

- This Release is a beta-version and may contain bugs, please report them
- It may only work with 2.BL as second league selected
- When you switched the second league in PESEdit Launcher the relinking might not work anymore, in that case you have to install again
