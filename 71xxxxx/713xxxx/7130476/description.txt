
PHP se está erigiendo como un lenguaje de programación muy potente enfocado a la creación de páginas web dinámicas. En este curso, válido para la versión 4.0 o superior, vamos a aprender desde cero a utilizar PHP para crear aplicaciones, cómo interaccionan PHP y HTML y cómo se puede aprovechar una de sus ventajas fundamentales: el uso de PHP para acceder de una forma sencilla a las bases de datos.
