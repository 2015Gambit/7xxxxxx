There are two kind of password adopted in the PDF files: user password and owner password.
* User Password
It is also called open password, and it is the password defined by the people who create the PDF file which is used for protecting the PDF files from opening by others. If a PDF file is protected with the user password, only people who enter the password can open the PDF file.
