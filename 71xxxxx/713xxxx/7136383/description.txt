Serious Sam HD: Double Pack (Jimbus edition).
---------------------------


Features of this version:
--------

- It includes Serious Sam HD Second and First Encounter (maps -Egypt- playable from the UI of the Second Encounter). So, it is like Second Encounter + the Fusion Pack.

- It is updated with the latest patch.

- Multiplayer and cooperative play available (with SmartSteam; everything included).

- Cheats and a trainer included.


What does this version unique? It allows you to play ONLINE WITH OTHER PEOPLE WHO HAVE ALSO DOWNLOADED AND INSTALLED THIS VERSION!!!

This version needs Steam and uses SmartSteam, which is what makes it playable online. Read the pdf file for detailed information. Everything is included.
