Es un conjunto de utilidades técnicas autoarrancables y gratuitas seleccionadas por este blog y unidas con Xboot. El conjunto está orientado principalmente al diagnostico de unidades de discos duros y memoria ram acompañado por aplicaciones para clonar y respaldar los datos de un disco duro, gestionar particiones, restablecer contraseñas de Windows y realizar un borrado completo y seguro de un disco.
Este conjunto de utilidades ha sido seleccionada para cubrir una necesidad personal de unificar en un mismo CD todas las aplicaciones habituales de uso diario para la reparación y diagnostico de algunos equipos informáticos.

¿What is HDD Diagnostics Tool?

It is a set of tools and techniques bootable free and bonded with Xboot selected. The set is intended primarily to diagnose hard drives and RAM accompanied by applications to clone and backup data on a hard disk, manage partitions, reset Windows passwords and make a complete, secure erasing a disc.
This set of utilities has been selected to fill a personal need to unite on the same CD all the usual applications of daily use for the repair and diagnosis of some equipment.
