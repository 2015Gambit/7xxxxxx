Record, edit, share.. You have a mini voice & music studio in your pocket!
TapeMachine is a high-quality sound recorder and editor, featuring a powerful waveform display.
- zoom and scratch
- select, export, edit, undo
- copy, cut, paste
- fade in/out, reverse, normalize
- looped playback
- time markers
- variable speed
- record when locked
- up to 2h/file
- input gain and auto-gain
- compact home screen widget
- bluetooth support
- powerful file manager with folders
- send by mail, dropbox, etc..
- use front or back mic (if available)
- set engine samplerate, latency, ..
- rec/play/convert WAV, AIFF, OGG, FLAC
