Depois que fizer o download, emule as imagens e execute o arquivo GrimSetup1.0.exe que o jogo irá se instalar. 

Funciona em qualquer OS Windows, incluindo Vista e Seven 64bit e 32bit. 

Quando terminada a instalação vá na pasta do jogo e clique com o botão direito em Grim Fandango Launcher.exe e execute em modo de compatibilidade para Windows NT/98. 

Depois execute o Launcher e vá em Options e clique em Run Grim From Hard Drive, depois é só clicar em run Windowed (para jogar em janela) ou Full Screen para jogar com a tela toda.

Obrigado.



After you finish the download, emulate the images and execute the file GrimSetup1.0.exe and the game will install. 

Works on every Windows OS including the Vista and Seven 64bit & 32bit.

When the installation is completed go to the game folder and right click in Grim Fandango Launcher.exe to execute it in the Windows NT/98 compatibility mode.

After that execute the Launcher and go to Options and left click in Run Grim From Hard Drive, then just click in run Windowed or Full Screen.
