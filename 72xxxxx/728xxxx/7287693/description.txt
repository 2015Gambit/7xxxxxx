ENGLISH:
This is a classic game from around 2000, it's created by Alawar.
Although this company rejects this game, I decided to release it here on Piratebay.
This game should work fine for Windows XP.

Problems starting this up can be solved easily for Windows 7.
Try starting it up in compatibility mode.
Right click the "PacMania.exe" go to Properties -> Compatibility -> Compatibility Mode -> Windows XP (Service Pack 3) -> OK
Although there shouldn't be a problem starting this up.

If you still encounter problems, you might have to try installing it from the installation.exe.
And again, Windows 7 users put this Installation.exe to compatibility mode, just like shown before.

DUTCH:
Dit is een klassieke spel van 2002, het is ontwikkeld en gepubliceerd door Alawar.
Omdat deze bedrijf het spel negeren, ben ik er van overtuigd dat ik het zal publiceren op Piraatbaai.
Dit game werkt sowieso met Windows XP.

Problemen met het opstarten van het spel voor windows 7?
Probeer het spel op te starten in compabiliteit modus.
Rechter muisknop op "PacMania.exe" go naar Eigenschappen -> Compabiliteit -> Compabiliteit Modus -> Windows XP (Service Pack 3) -> Toepassen, en dan op OK.
Maar er zal echter geen problemen zijn, als je het opstart.

Als je toch nog steeds problemen hebt, moet je misschien de Installation.exe proberen.
Je moet wel eerst dit ook in compabiliteit zetten, zoals we gedaan hebben hierboven.
