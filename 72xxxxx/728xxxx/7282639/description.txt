What is Exploit Pack?

Expoit Pack is a set of tools that Juan Sacco made to allow you easily test the security of remote targets find the weakness points and protect them. We use this tools on the penetration testing we do beside the manually test. This kind of tools comes in handy when you need to do a controlled pentesting . And of course, have technical support to ask questions when you get stuck.

What we do?

We develop software, we have two main tools one focused on web security assessment to let you control onbehalf of a remote browser all the possible actions that an attacker can do, this tool also allow you to  protect your site from SQLi, XSS, DoS and ban the attackers among other functionalities. We also have a tool to assist you while testing your desktop security by launching the latest exploits in the market, it has a module editor, update manager and other features.

How Exploit Pack will help you?

We can help all kind of users by giving the right tools to help them test the security or vulnerability of their implemented technologies. We are focused on our client. If needed we can personalize any functionality of our tools. We also provide custom penetration tests services in small and large sizes, these tests are performed by our best security professionals. Please consider us as a partner, we will love to help you.
