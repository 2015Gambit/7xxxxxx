Title: Europa Universalis
Genre: grand strategy / historical / real-time
Released: 2nd February 2001
Developer: Paradox Interactive
Publisher: Paradox Interactive
Description:
Europa Universalis is a historical strategy game that simulates all aspects of world history from 1492 to 1792. Each player takes on the role of one of the major nations of the era, controlling diplomacy, economy, warfare, exploration, and colonization.  The game takes place on a map divided into approximately 1,500 provinces, and proceeds in a pausable real time format.

Bonus content: manual.

How to play:
1) Download the game;
2) Mount or burn disc image;
3) Run Setup.exe to install (it will start automatically if you have autorun enabled);
4) Play the game without the disc using main executable or shortcuts.

! Notice
Windows 7/Vista users: before launching the game, navigate to where you installed it and turn on Windows 2000 compatibility for europa.exe.
