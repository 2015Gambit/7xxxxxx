NEW FEATURES 



Version 3.3.8 was released on May 12th, 2012:

1. Add "The New iPad" as supported output formats.

2. Fix the bug that fails to convert merged video clips to WMV or audio formats.

3. Improve the MP4 Faststart function to enable MP4 streaming.

4. Add Free Lossless Audio Codec (FLAC) as supported output profile.
