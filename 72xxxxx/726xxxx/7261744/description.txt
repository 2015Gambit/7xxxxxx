Title: Odyssey: The Search for Ulysses
Genre: adventure / puzzle-solving / 3rd-person
Released: 2nd January 2001
Developer: Cryo Interactive Entertainment
Publisher: DreamCatcher Interactive
Description:
Odyssey: The Search for Ulysses is an adventure game, with 3D characters moving in a pre-rendered environment. The game's universe stays faithful to Homer's Odyssey, including its characters, Poseidon, Zeus, Cerberus and the Cyclops. Penelope, who has had no news of Ulysses for many years, requests that Heritias set off in search of his childhood friend. Heritias first goes to Troy, the last place the leader of the Achaean troops was seen. In the game you have to fight creatures like the Gorgon and the Cyclops and you will have to foil the manipulation of the gods to escape from the conspiracy of Poseidon's agents, and resist the seductive enchantress.

Bonus content: manual.

How to play:
1) Download the game;
2) Mount or burn disc images;
3) Run Setup.exe to install;
4) After installation is complete, extract ODY101EN.zip archive to game folder to patch the game (overwrite old file);
5) Play the game with the second disc inserted.

! Notice
Windows 7/Vista users: before installing the game, set Setup.exe to Windows 2000 compatibility, otherwise you will get an error and the installation will not succeed. The game itself will run fine.
