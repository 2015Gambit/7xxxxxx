===============================================
Dynamic Disk Converter 3.2 Server Edition
===============================================

**** 1. Introduction ****

Dynamic Disk Converter is a powerful disk management software. Its main feature is to convert a dynamic disk back to basic disk without data-losing. It can ensure Operating System bootable after the conversion. 


**** 2. System Requirements ****

Dynamic Disk Converter 3.2 Server Edition can be worked perfectly on Windows 2000/XP/2003/Vista/2008 and Windows 7 (32/64bit) platform. To run Dynamic Disk Converter, the user must have the administrator privilege. Note: Windows 2000 system requires Service Pack 4.

Support for FAT12, FAT16, FAT32 and NTFS.
IDE, SATA, SCSI, USB external disk, and Firewire hard disk are all supported.
The maximum capacity of the single disk is 2TB.
