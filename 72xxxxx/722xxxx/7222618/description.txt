Microsoft Diagnostics and Recovery Toolset 6.5 en Español solo para Windows 7 es una herramienta booteable de Microsoft que nos permite ejecutar diferentes aplicaciones de recuperación para solucionar conflictos en Windows 7 y Windows Server 2008. Es similar al llamado disco de recuperación de Windows 7 que ya viene con las instalaciones de Windows 7 pero añadiendo las herramientas que incorpora la opción Microsoft Diagnostics and Recovery Toolset. Las 14 herramientas del conjunto de herramientas proporcionan muchas opciones de recuperación, incluso cuando no funcionan el modo seguro ni el inicio normal de Windows. El entorno de inicio sin conexión es fácil de usar. Pueden recuperar archivos eliminados y quitar malware de sistemas infectados mientras el equipo no está conectado. Esto ayuda a proteger a otros equipos de la red y reduce el tiempo durante el cual el equipo no está disponible.
