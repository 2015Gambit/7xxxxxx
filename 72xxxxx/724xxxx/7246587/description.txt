Based on RUU 3.08.401.1 HTC Evo 3D GSM X515m (SHOOTER_U)
Android 4.0.3 (Ice Cream Sandwich)
Kernel: 3.0.16 HTC Stock
Virtuous Dev Team: Diamondback, Flemmard, seo, rmk, chrisch1974, eViL_Dee, mdj, cypis, cjward, tbalden

Installation:
Die Installation in wenigen Schritten:
1. Installiere über das Hotboot die neueste Firmware (Hotboot? Schau ins "FAQ" weiter unten)
2. Nach der Installation der neuesten Firmware gehst du in dein Recovery
3. Ein Full-Wipe sollte nun von dir durchgeführt werden (Full-Wipe? Schau ins FAQ unten schauen)
4. Die Virtuous Shooter Rom über "install zip per SD Card" installieren.
5. Daraufhin die ganzen "Patches" installieren. Prinzip ist das selbe wie bei Punkt 4.
6. Das Handy neu starten. Der erste Start kann bis zu 6 min. dauern.
