This is a customized Ubuntu Linux Distribution based on Backtrack 5R2 - Gnome.

The additions are:
- GUI boot and login screen
- VLC
- Backtrack 5 Camo splash screen (Sorry you have to add your own wallpaper :( )
- Pandora client: Pithos
- OpenOffice.org
- Gnome-Games
- Gnome-Restricted-Extras
- Gnash Flash Plugin
- Synaptic Package Manager
- Galculator
- Texas Instruments Calculator Emulator
- Skype
- GIMP
- Compiz Settings Manager
- Remastersys: This is the package used to create the image
