YooTheme Widgetkit v1.0.5 for WordPress.

The following widgets are included:

Accordion
Gallery
Lightbox
Map
Mediaplayer
Slideset
Slideshow
Spotlight
Twitter

Installation instructions:

1. Download the torrent and unpack it. Rename the folder to widgetkit.
2. Using FTP, upload this directory to the /wp-content/plugins directory of your WordPress installation on your server.
3. Go to Plugins ┬╗ Installed Plugins in the WordPress administration and Widgetkit will appear in the overview of all available plugins.
4. Click on Activate to activate the newly installed Widgetkit plugin.
5. Enjoy
