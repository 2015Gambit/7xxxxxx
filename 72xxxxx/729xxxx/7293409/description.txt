Total Commander (former Wincmd) is a file manager for Windows similar to the Windows Explorer. But Total Commander uses a different approach: it has two fixed windows side by side like a well-known file manager for DOS.Features:
ΓÇó Two file windows side by side
ΓÇó Multiple language and Unicode support
ΓÇó Enhanced search function
ΓÇó Compare files (now with editor) / synchronize directories
ΓÇó Quick View panel with bitmap display
ΓÇó ZIP, ARJ, LZH, RAR, UC2, TAR, GZ, CAB, ACE archive handling + plugins
ΓÇó Built-in FTP client with FXP (server to server) and HTTP proxy support
ΓÇó Parallel port link, multi-rename tool
ΓÇó Tabbed interface, regular expressions, history+favorites buttons
ΓÇó Thumbnails view, custom columns, enhanced search
ΓÇó Compare editor, cursor in lister, separate trees, logging, enhanced overwrite dialog etc.
ΓÇó Unicode names almost everywhere, long names (>259 characters), password manager for ftp and plugins, synchronize empty dirs, 64 bit context menu, quick file filter (Ctrl+S)
ΓÇó New: USB port connection via special direct transfer cable, partial branch view (Ctrl+Shift+, and many improvements to ftp, synchronizing and other functions
ΓÇó And many more!
