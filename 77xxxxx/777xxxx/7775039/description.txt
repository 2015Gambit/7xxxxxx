RepaintMyImage

RepaintMyImage allows you to make your images painterly. 

You can use 3 different algorithms:
Smudge - soft curly repainting
Repaint - hard edge repainting
Cartoonize - comic style

Supported OS:

Windows 7/vista/xp

Homepage: http://www.adptools.com/

Installation Instructions:

1. Install program.
2. Use license.txt file to register application.
3. Done, Enjoy
