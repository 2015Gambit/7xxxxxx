               Hard Drive Inspector v4.0.137.0-Multi+PATCH- TPB-DeGun


Basé sur la technologie S.M.A.R.T, Hard Drive Inspector est un programme puissant et efficace qui vous permet de contrôler l’état de votre disque dur 
et de prévoir un éventuel « crash ».

Vous êtes ainsi mieux protégé contre la perte de vos données électroniques. Ce programme peut être aussi bien utilisé pour un usage privé que pour un 
usage professionnel d’administration réseau.

Fonctionnalités :

Contrôlez l’état de tous vos disques durs : Hard Drive Inspector peut gérer jusqu’à 32 disques durs.
Vérifiez toutes les informations S.M.A.R.T nécessaires : valeurs courantes, valeurs seuils, valeurs les plus hautes, valeurs les plus basses.
Visualisez les informations sur votre disque dur : capacité, nombre de cylindres, secteurs, taille de mémoire tampon, etc.
Vérifiez la température de votre disque dur s’il est équipé de capteurs thermiques.
Choisissez votre mode d’utilisation : périodique ou vérification au démarrage et à la sortie du système.
Soyez prévenu avant pour éviter le pire : le programme peut vous envoyer un message par email si un disque dur est en train de tomber.

Hard Drive Inspector v4.0.137.0
Hard Drive Inspector is an easy-to-use program that monitors computer disks' health to prevent crashes and expensive hard disk recovery. The value of 
information stored on an average computer hard drive often exceeds the value of the computer itself. But a hard drive is one of the most unreliable 
internal computer devices. When it crashes (which happens quite frequently), the stored information will be lost fully or partially. Subsequent hard 
drive recovery is either not possible at all, or constitutes a very sophisticated and expensive procedure that can be performed only by specialists.

Modern hard drives support S.M.A.R.T. (Self-Monitoring, Analysis, and Reporting Technology) that was specially developed to detect upcoming disk failures. 
Using this technology Hard Drive Inspector can monitor the current state of various life-critical parameters describing a disk's reliability and 
performance. 

Hard Drive Inspector analyzes this data and warns the user if the parameters indicate a critical reduction of disk reliability. 

Timely alerts make it possible to copy the data and replace an unreliable disk, thus preventing information loss and costly hard drive data recovery.

Platform: 2000/XP/2003 Server/Vista/Windows 7
