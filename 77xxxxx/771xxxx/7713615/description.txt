Process Lasso PRO 6.1.0.59 [Multi/PL]

System: Windows 2000, XP, 2003, 2008, Vista i Windows 7 [32/64 bit]

Process Lasso poprawia wydajność systemu, elastyczność i stabilność podczas dużych obciążeń dynamicznych poprzez korektę przetwarzania klas priorytetowych. Można również automatycznie usuniąć niepożądane procesy, zastosowanie priorytetów domyślnych i koligacje procesorów do procesów , i zapisywać wszystkie procesy. To nie wymiana menedżer zadań, ale ma menedżera zadań, jak zdolności pochwalić swoich podstawowych funkcji

Process Lasso improves system performance, responsiveness, and stability during high loads through dynamic adjustments to process priority classes. It can also automatically terminate undesired processes, apply default priorities and CPU affinities to processes, limit the number of instances a process can have, and log all processes. It is not a task manager replacement, but does have task manager like capabilities to compliment its primary functions.
