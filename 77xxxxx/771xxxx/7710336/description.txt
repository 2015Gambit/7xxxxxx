XWidget - a portable version of the new platform to change the look and functionality of the Windows desktop by installing a special gadgets you can decorate to your liking desktop and gadgets completely replace the system of Windows 7.


The main features of xWidget:
- Nice and easy to use interface;
- A lot of good ready-made customizable widgets;
- User-friendly graphical interface for configuring and adding widgets;
- Built-in editor widget that allows you to not only edit existing widgets, but also create their own;
- Low consumption of system resources.
