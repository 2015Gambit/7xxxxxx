                           INSTRUCCIONES

1) Crear una carpeta llamada GAMES
2) Crear dos carpetas llamadas TITUS y otra llamada MOKTAR
3) Descomprimir los archivos  en su carpeta correspondiente
4) Extraer el trainer ( Titus the fox trainer.zip) dentro de cada carpeta creada anteriormente
5) Instalar DOSBox0.75 dentro del directorio GAMES o donde quieras
6) Al ejecutar DOSBox0.75 veras una ventana donde deberas montar una unidad virtual de la siguiente forma:
6a) Escribir: mount c c:GAMES (donde GAMES es el directorio creado al principio)
6b) Para ir a la carpeta MOKTAR escribe cd MOKTAR
6c) Para ir a la carpeta TITUS escribe CD TITUS
7a) Una vez dentro de la carpeta deseada si deseamos ver los codigos de los niveles tipeamos:
FOX COD
7b) Si deseamos usar el trainer tipeamos:
FFC TRN
8) Si todo funciona correctamente podran utilizar las siguientes teclas cuando comienza el juego:
F5 - Energia infinita (Toggle unlimited health) (spikes etc. are still lethal)
F6 - Vida infinita (Toggle unlimited lives)
F7 - Modo NOCLIP utiliza las teclas para desplazar a Titus (Toggle noclip mode)

Dentro del archivo zipeado deben leer el archivo README con las instrucciones para utilizar el editor de niveles.
