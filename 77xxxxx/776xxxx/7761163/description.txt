System requirements : 
Processor: 1 gigahertz (GHz) or higher.
RAM: 1 gigabyte (GB) for the 32-bit version or 2 GB for 64-bit version.
Free hard drive space: 16 gigabytes (GB) for the 32-bit version or 20 GB for 64-bit version.
Graphics: the graphics device Microsoft DirectX 9 or later.

Additional requirements to use certain features
To use the touch features require tablet or monitor with multi-touch
To access the Shop Windows to download and run applications that require an active Internet connection and a screen resolution of at least 1024 x 768 pixels.
Applications need to bind with resolution of 1366 x 768
