This is the torrent of the Magic: the Gathering set definitions for OCTGN, from http://www.octgngames.com (http://octgn.gamersjudgement.com)

You can download the new OCTGN client at www.octgn.net

This torrent contains all expansions that have been released during Innistrad Block (excluding the actual Innistrad set, as that was included in our first torrent).

You can find the rest of the sets not included in this torrent here: http://octgn.gamersjudgement.com/viewtopic.php?f=8&t=194

The Magic: the Gathering Game Definition is required to play Magic.  You can find the most current version here: http://octgn.gamersjudgement.com/viewtopic.php?f=8&t=195

Wizards will often update the rules text of cards, known as the Oracle text.  When this happens, we will release a Patch to update the sets found in this torrent.  You can find the most current patch here: http://octgn.gamersjudgement.com/viewtopic.php?f=8&t=196

OCTGN is designed as an open shell for many different kinds of card games.  While Magic is the most popular, OCTGN has a large community of players and features dozens of different games.  You can find more information on other games at our official OCTGN game directory, http://octgn.gamersjudgement.com
