System Requirements

Acrobat XI Pro for Windows® requires the following:

·         1.3 Gigahertz or faster processor
·         Microsoft® Windows XP with SP 3 for 32 bit (with SP 2 for 64-bit); Windows Server® 2003 R2 (32 bit and 64 bit); Windows Server 2008 or 2008 R2 (32 bit and 64 bit); Windows 7 (32 bit and 64 bit); Windows 8 (32 bit and 64 bit)
·         512 MB of installed RAM (1 GB recommended)
·         1.9 GB of available hard disk space (Note: The installer creates temporary files during installation so the actual hard disk space required at that time is greater than the value listed.)
·         1,024 x 768 screen resolution
·         Internet Explorer 7, 8, 9, 10; Firefox (ESR); Chrome
·         DVD-ROM drive, if installing from a DVD
·         Video hardware acceleration optional

For Windows Server 2003 SP2 and Windows XP SP2 for 64-bit, Microsoft Hot Fix KB-930627 is required. See: http://support.microsoft.com/kb/930627
