Latest release of Bodhi Linux, a minimalist, Ubuntu-based, operating system. Uses the lightweight Enlightenment GUI. System requirements include 128 MB RAM, 1.5 GB hard disk space, and a 300 MHz processor.

Website: http://www.bodhilinux.com/

Appcenter: http://appcenter.bodhilinux.com/
The Bodhi Appcenter allows one to install supported applications and packages with a single click.
