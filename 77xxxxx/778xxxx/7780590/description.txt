 DESCRIPCIÓN

 Diseño. Desarrollo. Comprobación y publicación. Más rápido.

 Adobe® Dreamweaver® CS5.5 es la aplicación que lidera el sector de la edición y creación de contenidos web. Proporciona funciones visuales y de nivel de código para crear diseños y sitios web basados en estándares para equipos de sobremesa, SmartPhones, tabletas y otros dispositivos.

 



NOVEDADES

 Compatibilidad con CSS3/HTML5

 Aplique estilos con el panel CSS, que se ha actualizado para cumplir las nuevas normas CSS3. La vista de diseño ahora admite las consultas multimedia y aplica diferentes estilos a medida que va ajustando las dimensiones de la pantalla. Utilice el código del futuro con HTML5, las sugerencias de código y la compatibilidad con la representación de la vista de diseño. Ahora, la vista interactiva incluye compatibilidad con “vídeo” (QuickTime) y con etiquetas “svg”.

 Integración de jQuery

 Añada interactividad avanzada con las sugerencias de código de jQuery. jQuery es la biblioteca de JavaScript estándar del sector que simplifica la adición de una amplia variedad de interactividades a las páginas web. Agilice su trabajo con las plantillas de iniciación para teléfonos móviles.

 

Aplicaciones nativas Android e iOS con PhoneGap

 Cree y agrupe aplicaciones nativas para Android™ e iOS gracias a la nueva función PhoneGap. Convierta el código HTML actual en una aplicación para teléfonos móviles con Dreamweaver, mediante el marco de trabajo de código abierto PhoneGap.

 

Integración de Adobe BrowserLab

 Previsualice páginas web dinámicas y contenido local mediante numerosas herramientas de visualización, diagnóstico y comparación. Dreamweaver está integrado en Adobe BrowserLab, un servicio en línea de CS Live1,2 que prueba de forma rápida y precisa el contenido web en diferentes exploradores web y sistemas operativos.
