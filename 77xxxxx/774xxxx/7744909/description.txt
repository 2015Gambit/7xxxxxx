Pokémon Noir 2 (français)
C'est une version patchée.
Le jeu est au format nds.
Il est compressé en tar et 7z. Sous Linux et BSD : vous devez déja voir un utilitaire. Sous Windows : 7zip devrait faire l'affaire. Pour les autres, vous devriez facilement trouvé un logiciel, les 2 formats étant libres.

Le jeu a été testé sous Desmume 0.9.7, sous Linux (Ubuntu 12.04). 
http://desmume.org/download/
