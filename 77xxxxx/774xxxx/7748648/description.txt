Desenvolvido por SCYPT.

1. Execute o aplicativo no modo administrador
2. Selecione o sistema operacional.
3. Selecione o servidor.
4. Clique em "Activate", pode demorar um pouco para ativar dependendo da velocidade de sua internet.

Microsoft Windows 8
Microsoft Windows 8 Enterprise
Microsoft Windows 8 Professional
Microsoft Windows Server 2012 Datacenter
Microsoft Windows Server 2012 Standard

A ativação é válida por 180 dias, antes de expirar reative o sistema.

Eu tenho as keys de todas as versões, mas só liberei neste aplicativo as que eu testei. Em versões futuras irei liberando as outras keys.

Caso não consiga ativar, altere seus servidores DNS para os do Google (8.8.8.8 e 8.8.4.4)
