English

* Description *: In just over three years since its launch, Google Chrome has already reached its 22th version.

As is usual in the browser updates, no significant change was implemented in relation to the latest version, only bugfixes and some technical enhancements.

The major focus of the last update is improving the browser for Windows 8, minor improvements for Retina display screens and a new feature that lets you hide the mouse pointer in JavaScript environments. The old screen that showed the browser version was substituted, giving the space a tab to the same function.

Furthermore, there was a change visually perceptible, in which the former button tools in the form of a wrench, was replaced with "three lines". Although in terms of operation it does not represent a change, the old icon remained the same since the first version of the browser.

Português(Brasil)

*Descrição*: Em pouco mais de três anos desde o seu lançamento, o Google Chrome já chega à sua 22ª versão.

Como é de costume nas atualizações do navegador, nenhuma mudança significativa foi implementada em relação à última versão, apenas correções de bugs e alguns aprimoramentos técnicos.

O grande foco da última atualização é o aprimoramento do browser para Windows 8, pequenas melhorias para telas Retina display e um novo recurso que permite ocultar o ponteiro do mouse em ambientes JavaScript. A antiga tela que demonstrava a versão do navegador foi substituída, dando espaço a uma aba para a mesma função.

Além disso, houve uma alteração visual perceptível, na qual o antigo botão de ferramentas, na forma de uma chave inglesa, foi trocado por “três linhas”. Embora em termos de funcionamento ela não represente uma mudança, o antigo ícone permanecia o mesmo desde a primeira versão do navegador.
