A Collection is a six compact disc box set The Doors, released by Elektra and Rhino on July 5, 2011.
This collection packages the complete anthology of The Doors albums before Jim Morrison's death in 1971. It features the 40th anniversary versions of the tracks, 
remastered by Bruce Botnik.
