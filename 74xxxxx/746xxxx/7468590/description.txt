Need For Speed - Porsche Unleashed

SYSTEM REQUIREMENTS
MINIMUM PC REQUIREMENTS
MINIMUM
Windows 95/98
200MHz or faster Intel Pentium MMX, or AMD K6 processor
4X CD-ROM/DVD-ROM drive (600K/second transfer rate) using 32-bit Windows 95/98 CD-ROM/DVD driver
150MB free hard disk space
32MB RAM
4MB PCI or AGP video card with full Direct3D support and DirectX 7.0a compatible driver
DirectX 7.0a compatible sound card, mouse, keyboard
RECOMMENDED
300MHz or faster Pentium processor
64MB RAM
8X or faster CD-ROM/DVD-ROM drive (1200K/second transfer rate) using 32-bit Windows 95/98 CD-ROM/DVD driver
625MB free hard disk space
8MB PCI or AGP video card with full Direct3D support and DirectX 7.0a compatible driver
MULTIPLAYER SYSTEM REQUIREMENTS
INTERNET
56.6kbps of faster internet connection.
1 CD per computer.
NETWORK
TCP/IP or IPX/SPX compliant network
1 CD per computer
MODEM
100% DirectPlay compatible 56.6kbps or faster modem
High speed serial port (16550 UART) for external modem
1 CD per computer

CONTROLS
DEFAULT COMMAND KEYS
MOVEMENT
Arrow Up	 accelerate
Arrow Left/Right	 steer left/right
Arrow Down	 brake
Arrow Down(Press and Hold)	 reverse
EXTRA CONTROLS
Spacebar	 handbrake
A/Z	 shift up/down
H	 horn
R	 reset car
VIEWS
C	 cycle camera views
B	 look behind
Y/I	 look left/right
LIGHTS
L	 headlights
T	 hazard lights
W/E	 turn signal left/right
HOT KEYS
Escape	 pause race
F2/F4	 rotate pause camera right/left
F3	 stop pause camera rotation
F5	 toggle on-screen gauges on/off
F6	 toggle review mirror on/off
F7	 toggle map on/off
F8	 toggle translucent hud on/off
F9	 toggle music on/off
F10	 toggle sound effects on/off
F11	 cycle view distance
F12	 toggle horizon on/off
Alt + P	 take a screen shot
