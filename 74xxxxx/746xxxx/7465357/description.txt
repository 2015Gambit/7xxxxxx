Sinopse: 

Ammyy Admin é um pequeno programa para você administrar computadores à distância, a partir de uma ID matriz, que de certo modo trabalha como um roteador. Útil para manter sob controle o conteúdo acessado por funcionários dentro de uma empresa, saber para que fins seus filhos estão utilizando a Internet, ou até mesmo para que você possa consertar o computador de algum cliente seu sem precisar buscar a máquina. 

Basta que você baixe o programa em ambos os computadores que deverão fornecer trocas de informações: o do administrador e o do administrado (o seu e o do seu filho, por exemplo). Então é preciso trabalhar com as abas Cliente e Operador. Para isso, forneça sua ID (é possível que o Ammyy Admin faça tal reconhecimento automaticamente). Clique em começar e aguarde alguns segundos até ficar conectado ao roteador. Para desligar, mais tarde, é simples: clique em Parar.

Se você pretende mexer com a aba Operador, então é necessário pegar a ID dos outros computador e preencher o espaço "Cliente ID". Abaixo, você configura a velocidade de conexão, informando se é a mais baixa, média ou super rápida (opções com menos de 256kb até mais de 1024kb). 

Dentro das Settings (configurações) do software, poucas alterações podem ser feitas, tais como requisitar o roteador do cliente e avisar ao alterar para o modo tela cheia (opções para o modo Operador), ou remover o plano de fundo da área de trabalho e capturar hints (opções para o modo Cliente). Independentemente da linguagem que você esteja utilizando para Ammyy Admin, as configurações sempre estarão em inglês.
