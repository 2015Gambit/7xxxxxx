
 ExtremeCopy Pro to narzędzie do przyspieszania procesu kopiowania plików. Aplikacja znacząco skraca czas potrzebny na operacje dyskowe (szybkość kopiowania plików zwiększa się od 20 do 120%). ExtremeCopy Pro posiada również możliwość wstrzymywania i wznawiania procesu kopiowania. Program zastępuje standardowe okno kopiowania w Windows

 Support Operation System: XP / 2003 / VISTA / VISTA64 / Windows 7 32bit / Windows 7 64bit
