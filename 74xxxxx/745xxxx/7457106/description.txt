Trine 2 v1.16 (Steam/non-Steam)
-------------


Features of this edition:
--------

  - The game is complete and updated to 1.16 version.
  - Every mission can be unlocked.
  - It includes tutorials, a trainer and the libraries the game requires.
  - The game can be played online using "Steam" (neither "GreenLuma" nor "SmartSteam" are needed) or offline.


Notice this is not an update but the full game. Do not install it over an existing installation.
