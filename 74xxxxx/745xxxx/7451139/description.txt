Screenpresso - es ist ein einfaches Programm für das Betriebssystem Windows, so dass Sie 
tolle Aufnahmen von Ihrem Bildschirm. Screenpresso fängt die Druck-Taste, und macht die 
besten Bilder vom Bildschirm für einige Sekunden. Erlaubt es, Fenster und Bereiche auf dem 
Bildschirm zu erfassen, Scrollen des Bildschirms, bearbeiten Sie die aufgenommenen Bilder, 
Effekte hinzufügen, ändern Sie die Größe von Bildern, Screenshots schicken per E-Mail und 
sie auf Web-Ressourcen.
************************************************** ********************
Screenpresso ist eine Anwendung, die Ihnen helfen Schnappschüsse und Videos auf Ihrem 
Computer. Diese Software kann auch als portables Produkt installiert werden. In diesem Fall 
werden keine Änderungen an Ihrer Windows-Registry-Schlüssel vorgenommen werden. 
Darüber hinaus können Sie Screenpresso auf einem Wechselmedium speichern und direkt 
auszuführen seiner ausführbaren Datei auf einem beliebigen Computer.
Die Oberfläche der Anwendung ist sauber und intuitiv. Screenpresso leicht aus dem System-Tray zugegriffen werden. So können Sie erfassen einen Bereich, vorherige Region, Vollbild 
oder mit Auto-Scroll. Plus, können Sie einen eingebauten Editor zu öffnen, um Pfeile zu 
zeichnen, Rechtecke, Ellipsen oder eine Lupe, eine Region zu verwischen, fügen Sie ein 
Textfeld, Einfügen von Text in einer Blase, sowie Bilder hinzufügen, Nummerierung und 
Highlights.
Features:
- Nehmen Sie den gesamten Bildschirmbereich, ein Fenster mit einem Klick
- Nehmen Sie High-Definition Video
- Teile des Bildschirms Blättern Sie durch den Mechanismus der Adhäsion
- Saubere transparenten Ränder und Ecken auf Vista und Windows 7
- Hinzufügen eines Wasserzeichens
- Speichern Sie Screenshots und Videos nach Datum sortiert
- Ziehen Sie die Maus aus der Geschichte der Screenshots direkt in allen Programmen, die mit 
Bildern oder Video arbeitet
- Export von Video-Formaten MP4 (H264), WebM, ogv, Wmv, HTML 5
- Per E-Mail senden. E-Mail: leicht zu Dateien an E-Mail-Clients wie Outlook, Thunderbird, 
GMail (direkter Einschub) befestigen
- Senden auf YouTube, Twitter, Facebook, Twitpic, FTP-, Flickr-, G-Mail, Minus, Dropbox, 
Imgur, ImageShack
- Senden Cloudpresso
- Zu Evernote senden (funktioniert auch, wenn Evernote ist nicht installiert!)
- Durchführung von eigenen Scripts
- Der Editor basiert auf Vektorgrafiken basieren
- Bild-Effekte: Schatten, Rahmen, Reflexion, Hintergrundfarbe, Drehung
- Zeichenwerkzeuge: ein Pfeil, Rechteck, Ellipse, Nummerierung, Textfelder, Sprechblasen, 
Anmerkungen ...
- Malerei von vertraulichen Daten
- Erfordern keine Installation, erfordert keine Administratorrechte
