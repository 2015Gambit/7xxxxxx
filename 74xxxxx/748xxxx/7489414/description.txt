Just Checking for Windows is an easy-to-use program for managing checking accounts. The interface resembles a checkbook register. Each transaction details the date, pay type, check number, payee, memo, transaction amount, balance, and the date the transaction cleared the bank.

Just Checking for Windows Features

Manages multiple checking accounts.
Calculates actual and bank balances.
Supports multiple users.
Password protect your files.
Print checks.
Multi-Level sorting. Ascending and Descending sorting.
Advanced filtering.
Search the checkbook register. Case-sensitive and partial match.
Edit check templates.
Nine reports.
Custom Pay Types
Import Quicken and Microsoft Money files.
Set up reoccurring transactions.
Backup and Restore database.
Customize font and color.
Check for program update command.
Works on Windows 7, Vista, XP.
