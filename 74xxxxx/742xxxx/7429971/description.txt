Jack Slate, an innocent cop sentenced to death, makes a daring escape from the Iron Point Maximum-Security Penetentiary. Driven to bring the criminal element that framed him to justice, Slate hunts the hunters in this gritty, crime-noir inspired epic. With super-charged game play, jaw-dropping graphics and hard-boiled storytelling, Dead to Rights raises the bar on the crime action-thriller genre.

Release Date: November 11, 2003
Ratings     : M for Mature: Realistic Violence
Genre       : Third-Person Action
Publisher   : Namco
Developer   : Hip Interactive

For More Info Visit - http://www.allgame.com/game.php?id=43838

How to install

Run dead2rights application file... and it will be installed on selected drive, now go to the install directory and double click dtrsetup. go to graphic settings and adjust screen resolution according to your moniter. i use 800 by 600. now play game by clicking dtrpc.
