3- Run this inTerminal to flush the cache: sudo dscacheutil -flushcache

4- Install InDesign CS5 ME, use any of these serials:
1037-0738-2634-5894-1142-5282
1037-0587-4308-5833-4507-1241
1037-0946-0591-5042-0155-7084

This will install in Greek language. Follow the next steps to change from Greek to any other language.

5- Run ChangeIDmenuLanguage.app

6- Go to the InDesign CS5 folder in 'Applications'

7- Go to 'InDesign Shortcut Sets' and 'InDesign Workspaces' folders and duplicate the 'el_GR' folder in there.

8- Rename the folders you duplicated to the language you need, en_US is for English. See the list of available language names in the 'Legal' folder.

9- Open InDesign, then in the system menu click Window/Workspace, try the workspace you like, the 4th is the default.

10- Enjoy!
