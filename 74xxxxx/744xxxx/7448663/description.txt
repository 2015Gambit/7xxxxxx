Build 6.0.3.368 (July 16, 2012) release notes:
- added German and French language
- added support for Opera 12, Firefox/Thunderbird 15 and Seamonkey 2.12
- added option to start Sticky Password minimized to systray
- added context menu with Cut/Copy/Paste/etc. commands in Secure Memos editor
- added drag & drop for moving accounts between groups
- simplified Configuration Wizard after first installation
- password preview icon in Password Generator now toggles the preview permanently
- fixed duplicate hotkeys for Quick Launch and Lock
- fixed groups always being expanded after restoring the Main Database View
- fixed issues with "liking" on Facebook
- fixed support for autofill on many website


Languages: English, Russian, Spanish, Czech, German and French
