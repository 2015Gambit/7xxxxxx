Once again, like promised, M3 Software returns with another release.
It's been like 2 years since the last time we introduced something to the scene, so we thought we could finally take advantage of the time we have to release what people wanted us to release the last time....

Need For Speed Most Wanted Romanian Language Edition

The game is translated from English into Romanian - in terms of text.
It's an official M3 Software - 3ogdy release.

The system requirements remain the same as the game has NOT been modified in any other way than Language-wise.
It was tested on Windows XP x64 and Windows 7 x64 and it works flawlessly.
We even started the game from the beginning (yeah we had already installed Underground 2 Romanian Language Edition) and we got a bonus of about 10 grand in cash so we could start with the VW Golf instead of the SS...)

N-Joy another M3 Software - 3ogdy release and...don't forget... we've got another goldie coming out either today or maybe next week...



Call of Duty 2 Romanian Language Edition is on the way...stay tuned these days...

Pentru toti romanii carora le plac jocurile cu masini din seria NFS, M3 Software vine cu o noua lansare dupa succesul avut de Call of Duty 1 Romanian Language Edition si Need For Speed Underground 2 Romanian Language Edition.
Vorbim de NFS Most Wanted, jocul lansat in 2005 de catre EA Games.
Este versiunea in limba romana a jocului, din punct de vedere textual.
Cerintele de sistem raman aceleasi dat fiind faptul ca nu am modificat nimic altceva in joc...ci l-am pastrat original.

As vrea sa multumesc tuturor celor care mi-au lasat comentarii despre parerile lor in legatura cu lansarile anterioare si bineinteles ca multumesc anticipat celor care lasa un comentariu si pe aceasta pagina.

2012 va vedea si Call of Duty 2 Romanian Language Edition lansat in curand...(in cateva zile).

Jocul functioneaza in Windows XP x86/x64, Se7en x86/x64.


Montati imaginea si executati manual(daca nu se executa automat) fisierul AutoRun.exe.
Veti vedea ca de la programul de instalare pana la jocul in sine totul este in limba romana ca text.

Este un joc la care s-a lucrat destul si as aprecia daca ati lasa un comentariu cu parerile voastre despre aceasta lansare.
Jocul nu este nou, dar in comparatie cu CoD1 si NFS UNDRGND2, vorbim de ceva mai recent.
