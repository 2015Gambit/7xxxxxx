PARA INSTALAR:
1: Remova qualquer cartão de memória do seu GPS
2: Conecte o seu GPS no computador pelo cabo USB de Dados
3: Abra o GPS no modo "explorar" (como se estivesse abrindo um Pen-Drive para ver seus arquivos)
4: Apague todos os arquivos do GPS (ou mova eles para uma pasta do seu computador para fazer um backup)
5: Copie as duas pastas disponíveis neste diretório: IGO8.3 e SYGIC - para o seu GPS.
6: "Ejete" o GPS do computador clicando no botão para remover o dispositivo com segurança.
7: Ligue seu GPS e veja se está abrindo o sistema de navegação... note que em alguns casos é necessário espeficicar antes a localização do software IGO.
Caso isso ocorra, ao ligar o GPS clique em "AJUSTES" >> "ORIGEM DE NAVEGAÇÃO" e selecionar o arquivo "iGO8.exe" dentro da pasta IGO8.3
