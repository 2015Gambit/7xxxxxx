
SharpPlus Sqlite Developer is the name of the software is designed to manage SQLite database. SQLite is a small database (volume about 500 KB), which translated into a C library (Library) is written and the RDBMS database considered. This database without a license (Public Domain), free and open source is released. The advantage of a single database file size is small and independent of the operating system. Environment with good management covering all of its features. The only problem is the lack of database support for all SQL standard features that it is a big weakness. By this software, you can edit different parts of a database management practices necessary to easily do.

Some software features SharpPlus Sqlite Developer:
Powerful SQL editor with 
autocomplete feature automatically Kamlsazy 
support utf8 and utf16 
data generated output to sql, csv, excel, word, html, xml 
data from csv, sqlit 
LiveUpdate feature updates
