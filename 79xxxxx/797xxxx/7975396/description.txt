This game is a faithful remake of classic 89' action game - Wings Of Fury.

What's new in version 3.4:
   * new music tracks
   * achievements
   * lots of bug fixes
   * rocket bunkers
   * new levels
   * graphic tweaks
   * new languages: Italian, PT (Brasil), Catalan, Swedish, Hebrew (beta)

Enhanced version:
   * new units (submarines, flak cannons, Japanese bombers)
   * new levels
   * automatic license download

Runs under Windows XP/Vista/7 (32/64 bit)
