Extensions for WYSIWYG Web Builder


Can be used with 8.5.3 and older

 
http://www.wysiwygwebbuilder.com/extensions.html



Instructions:

1.In some of the files u will see 2 files .wbx and .uwbx and some only one(.wbx).
2.Open your WYSIWYG Browser
3.Click on the Extensions tab
4.Select install 
5.If u have 2 files(.wbx & .uwbx) then install both.
6 That's All. 



If U read the pdf files in the specific folders, 
U will see a better description of the extensions and the 
standard instructions to install it.
