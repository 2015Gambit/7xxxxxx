Vectir v2.7.1.2
Vectir is a smart and user friendly PC Remote Control application. With support for a range of devices and applications, Vectir is an ideal home and business solution. Use your mobile phone to control PowerPoint presentations at work and then control your Media Player when you get home.

Start controlling your PC straight away with pre-loaded support for Windows Media Player, iTunes, Winamp, PowerPoint, Mouse and Volume. Vectir also makes it easy to adjust existing behavior suited to your own personal requirements. Change font sizes and colors, add new commands and choose which keys on your phone control selected features. With the help of keyboard commands, you can add basic remote control support for virtually any application running on your PC!

Remote Desktop support is also provided by Vectir. This feature lets you remotely view and control your computers desktop from a mobile phone via Bluetooth or WiFi. Use the phones navigation keys to move around desktop, zoom in and out and control the mouse.

A wide range of phones are compatible with the Bluetooth Remote Control feature. Nokia, Sony Ericsson, LG, Motorola, Samsung, Benq Siemens, HTC and Blackberry phones have all been successfully tested. As long as your phone has Bluetooth or WiFi connectivity and runs Android or supports Java apps, there is a good chance it will work.

Features
* Bluetooth Remote Control
Control your PC from a Bluetooth enabled mobile phone.

* Application Control
Pre-loaded support for a number of popular software applications such as Windows Media Player, Winamp, iTunes and PowerPoint.

* Remote Profiles
Remote profiles let you interact with PC applications from your mobile phone. Browse playlists, view song information, skip tracks, change slides...

* Remote Profile Visual Designer
Design your own customised remote profiles for your mobile phone. Load your own skins, set customised colours, etc..

* IR Remote Control
Control your PC from any standard IR remote control.

* System Control
Gain full control of your operating system including, volume, mouse, keyboard and system commands such as shutdown.

* Remote Desktop
Remotely view and control your computers desktop from your mobile phone. Zoom, pan and mouse commands cursor commands are supported
