KMSnano v(7.2) Activator for win 8 & Office2013 | 64 Mb


This is a Key Management Service (KMS) Host Portable for Clients Windows 6.0, 6.1, 6.2, Office 14, 15. This is a virtual machine for run portable in windows, wich activate Volumen License version of Clients Windows and Office.

Change Log:

v7.0

- Super Faster Boot of the VM.
- Small.
- Code of KMS GUI improved.
- Add the automatic change of this values on Regedit: VLActivationInterval, VLRenewalInterval, NotificationDisabled, Manual.

Requierements: NET 4.0 or Windows 8

================
[INSTALL NOTES]
================

1. Double-click on KMSnano.exe.
2. Wait for the files to decompress.
3. Wait for activation process to finish.
4. Done.
