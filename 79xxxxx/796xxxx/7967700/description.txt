YUMI- Allows you to put multiple distributions including Windows XP & vista/7/8, all current Linux distributions, Hiren's Boot CD 15xx, etc. 


Instructions 
You will need a usb drive or a hard drive partioned and formatted as FAT32
How to Create a MultiBoot USB Flash Drive
Run* YUMI-0.0.8.3.exe following the onscreen instructions 

Run the tool again to Add More ISOs/Distributions to your Drive or press yes when asked to add other distributions at the end of the wizard.
Restart your PC setting it to boot from the USB device
Select a distribution to Boot from the Menu and enjoy!
