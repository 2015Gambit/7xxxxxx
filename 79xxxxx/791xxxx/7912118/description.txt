StoreTracer

StoreTracer is a program for self storage companies with up to 500 storage units. It is written in Visual C++, the software development language which is used for most serious Windows applications.The Windows style design ensures that the software is user friendly and transparent.

System requirements:

PC with Windows 98, Me, 2000, NT, XP, Vista, 7 (including 64 bit) and 8.
Minimum of 16 MB RAM.
Minimum screen resolution 800 x 600.

Homepage: http://www.grensoft.co.uk/StoreTracer/index.htm

Installation Instructions:

- Install Program.
- Use given serial to register application.
- Done, Enjoy.
