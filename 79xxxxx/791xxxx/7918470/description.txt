Theme manual:

* Apply a theme : menu-themes-choice theme-press apply button.
* Main wallpaper change : menu-wallpaper-Go wallpaper-choice one-press "set wallpaper" button.
* App drawer wallpaper change : menu-preferences-visual settings-backgrounds-app drawer background-Go theme background-choice one-press set background button.
* dockbar change : menu-preferences-visual settings-backgrounds-Dock background-Go theme background-choice one-press set background button.
* Large icon : menu-preferences-visual settings-Icon-Icons size-Choice large icon
* Icon Arrangement change : menu-preferences-screen settings-Grid size-choice one (screenshot is 5*4)
* icon label Transparency change : menu-preferences-screen settings-app icon labels- choice "show labels (transparent background)"

Required Android O/S : 2.0+
