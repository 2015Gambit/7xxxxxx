HireMate

HireMate is a program for small to medium sized car rental companies with fleets of up to 200 vehicles. It is written in Visual C++, the software development language which is used for most serious Windows applications. The Windows style design ensures that the software is user friendly and transparent. 

System Requirements:

PC with Windows 98, Me, 2000, NT, XP, Vista, 7 (including 64 bit) and 8.
Minimum of 16 MB RAM.
Minimum screen resolution 800 x 600.

Homepage: http://www.grensoft.com/HireMate/index.htm

Installation Instructions:

- Install Program.
- Use given Serial to Register.
- Done, Enjoy.
