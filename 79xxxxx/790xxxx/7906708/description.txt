StartIsBack

StartIsBack is a lightweight application that allows you to return the standard Start Menu and Start button to Windows 8.

If you cannot get used to the new Metro Start Screen, just use StartIsBack and bring the Start Menu back!

Supported OS:

Windows 8

Homepage: http://www.startisback.com

Installation Instructions:

- Install program
- Its pre-cracked, no need of any key.
- Done, Enjoy.
