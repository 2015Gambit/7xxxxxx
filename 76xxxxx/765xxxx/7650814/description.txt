Using the DRG Files
===================
-With offical I-Doser, this does not apply to Cracked I-Doser.

-Dose Lock-
Once you open a Dose in I-Doser, it adds a piece of text to the end of the file.
Open DRG files in a Text Viewer (Notepad) to view text:

"[Lots of text that makes up the dose]
 @[The unique code for your computer (about 11 characters long)]"

By removing the "@" and the text after it you unlock the file for any computer.
Once you open the Dose again, the text will be added back on to the end of that file, locking it to work on only your computer.


-What to do before using the doses-
Copy the Doses form the "DRG" folder to a New Folder, then only use the Doses in the New Folder.
This is so that you always have unused versions to share with your firends or to use on other computers.
