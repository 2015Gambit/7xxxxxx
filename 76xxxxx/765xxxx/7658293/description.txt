Windows Embedded POSReady (Formerly Windows Embedded for Point of Service) is an operating system developed by Microsoft as part of its Windows Embedded line of products designed for use in "POS Devices such as Automated teller machines and Self Service Checkouts.

Based on Windows XP Service Pack 3, this version offers more features over Windows Embedded for Point of Service V1 such as Full Localization and XPF Support if .NET 3.5 or higher is installed. It is the first version of Windows Embedded that can use the Windows Update Agent to update an installed and deployed image. Mainstream support will end in April 2014 and extended support in April 2019.
