SliTaz is a free operating system providing a fully featured desktop or server inless than 35 MB.

In Live mode SliTaz can run completely in RAM and boot from removable media such as a cdrom or USB key. The system is secure, stable and easy to use.


SliTaz "Cooking" LiveCD is a constantly evolving version, fully usable, it allows you to test the latest improvements and packages, ask for new features or report bugs to help us to prepare for the stable version. 

The ISO Image is bootable.

Seeding from 2 dedicated servers (2x 300mbit connection).
