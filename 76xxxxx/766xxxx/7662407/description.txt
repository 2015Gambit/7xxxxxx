This package contains helpful information on how to build your own home arcade cabinet from many sources.


Contained Files:

* The 2nd edition of the book Project Arcade

* The companion CD-ROM to Project Arcade

* Ultimate Arcade II blueprints, tools list, materials list, template and assembly  instructions in Word and PDF formats.

* Instruction for how to assemble an X-Arcade control base if you wish to purchase the controls and base.

* A PDF file on how to hide windows for a more authentic arcade look.

* A word document with links to many helpful sites for information and materials for building a home arcade cabinet.
