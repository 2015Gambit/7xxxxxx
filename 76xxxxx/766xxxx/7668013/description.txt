Tiny Windows Borders allows you to change szie of huge windows borders in Windows 8 RTM/RP.

It was created for those who prefer tiny windows borders, because Microsoft has removed user interface for appearence.

Move the slider to left if you want to disable huge windows borders in Windows 8, or set any border size from 1 to 10px as you like.

Compatible with Windows 7.
