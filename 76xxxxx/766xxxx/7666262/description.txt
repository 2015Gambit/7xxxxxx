Install Info:

Just run the EXE setup and install ConvertXtoDVD
(don't let ConvertXtoDVD run after install)

Click on only 1 of the Blue button's (Reg. #1, Reg. #2, Reg. #3, or Reg. #4) to register ConvertXtoDVD they automatically register ConvertXtoDVD for you each button registers with a different Serial so if one don't work try another
(make sure ConvertXtoDVD is closed before you click on one of the buttons) 

Now you are done close the setup and test out the program 

Have fun
