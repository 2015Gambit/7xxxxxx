'Vertical'

Vertigo/DC Comics, 2003

Written by Steven T. Seagle

Pencilled by Mike Allred

Inked by Philip Bond

Colored by Laura Allred


The original size of this book was, obviously, presented in a vertical format, but it also had a very narrow width. As such, you need to shrink it down to about 1/3 screen size if viewing on a full-size monitor. When you do that it looks great!
