Me complace compartir con vosotros la segunda version del que he denominado ASI MULTIBOOT USB, una recopilacion de sistemas y programas arrancables desde un lápiz usb, creado completamente por mi y destinado principalmente a administradores de sistemas informáticos (de ahí las siglas A.S.I.). 
He incluido en el las que creo que son herramientas imprescindibles para cualquier administrador de sistemas.

Concretamente:

-Acronis Backup & Recovery Server Universal Restore (Restaurador de sistemas windows con compatibilidad en hardware diferente).

-Acronis True Image Home 2012 Plus Pack (Hace lo mismo que Acronis Backup & Recovery, aunque mas limitado).

-Hiren's BootCD (Recopilacion de herramientas ms-dos muy útiles, incluye un Windows xp live).

-EASEUS Partition Master (Gestor de discos duros, para particionarlos etc, etc).

-Active Password Changer (Reseteador de contraseñas de sesiónes Windows).

-Windows 7 Portable (Como su nombre indica es un Windows 7 que se ejecuta desde lápiz directamente en cualquier PC, sin necesidad de instalarlo. Esta totalmente en español. Se recomienda meter en el lápiz herramientas portables al gusto, pues se ejecutan perfectamente. Ideal para recuperar sistemas que no arrancan y acceder a sus archivos o conectarse a internet, pues e añadido drivers para tarjetas de red incluyendo inhalambricas).

-Windows 7 Service Pack 1 (Instalador de Windows 7. Se incluyen versiones de 32 y 64 bits en Ultimate y Lite, esta ultima especial para ordenadores con bajos recursos, por ejemplo netbooks. Le he dado soporte para instalarlo desde un puerto USB 3.0, cosa no posible de manera oficial. Una vez instalado hay que activarlo con el activador incluido en la carpeta "Activador Windows 7-2008 R2").

-Windows Server 2008 32 Bits Service Pack 2 (Instalador).

-Windows 2008 Server R2 64 Bits Service Pack 1 Standard (Instalador. Tambien requiere activarse una vez instalado con el activador incluido en la carpeta "Activador Windows 7-2008 R2". Incluye soporte USB 3.0).

-Windows XP Profesional Service Pack 3 (Instalador. Con drivers SATA integrados y updates a Enero de 2012).

-Wifiway (Sistema destinado a la auditoria de redes wifi).

-Ubuntu (Instalador y portable).

-Beini (Otro sistema también destinado a la auditoria de redes wifi).

Todos los programas están en sus ultimas versiones y en español, a excepcion de aquellos que solo existen en ingles.
