Install Reckless Getaway v1.0.3.apk
Open and close Reckless Getaway
Place com.polarbit.Getaway in sd/data/data
Run Reckless Getaway again
If it does not work on your device, try deleting com.polarbit.Getaway and downloading it over wifi.

If it starts in Spanish, select "Ayuda y Opciones" then "Seleccione idioma"

Enjoy :)
