win 8 installeren:

Run setup.exe 
Gebruik deze key als er om wordt gevraagd: J7JBD-2NTPH-HXFHK-3FXFP-TMQG7
Als deze niet werkt, zoek op google en je zult er vanzelf meer vinden. ;)

Volg de stappen in de installatie en wacht een paar minuten voordat alles klaar is. 
(advanced mode = geen hotmail account, geen advanced mode = moet je een hotmail accont opgeven, en daarmee ook inloggen.)


win 8 branden op CD

kan waarschijnlijk (niet getest) hiermee: http://www.microsoftstore.com/store/msstore/html/pbPage.Help_Win7_usbdvd_dwnTool
of je kan hem met programma's als nero en imgburn branden op CD. 



Om windows te activeren is het handig dat je Skype hebt.

Volg deze stappen:

1. Disable je internet
2. Ga naar configuratiescherm > systeem en beveiliging > systeem > windows activation
3. Klik op "activeren"
4. Als het goed is komt er nu een error en kun je selecteren om via de telefoon te activeren
5. Klik op "Telefonisch activeren" en selecteer "United Kingdom" als land, hierdoor heb je meer kans dat hij succesvol activeerd.
6. Volg de instructies aan de telefoon, gebruik skype om een Toll-free nummer te gebruiken (kost geen geld)
7. Registreer je nieuwe code en je windows 8 is activated!


Als het je niet op deze manier lukt kun je altijd verder zoeken op google!
