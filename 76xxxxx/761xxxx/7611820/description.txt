Full Tekken 3 for PC with Emulator.

How to Configure ePSXe:
1: Open ePSXe.exe.
2: If the ePSXe Configuration window is not shown for the first time,then Click
on Config>Wizard Guide.
3: Click "Config>>" button.
4: Click "Next>>"
5: An error message will be shown,click ok and forget it.
6: Select "D3D renderer 1.11" & click "Next>>".
7: Select "ePSXe SPU core 1.5.2" & click "Next>>".
8: Select "ePSXe CDR WNT/W2K core 1.5.2" & click "Next>>".
9: Then Configure the controllers as you want & click "Next>>".
10: Click "Done"

How to play Tekken 3:
1: Click File>Run ISO
2: Select the "Tekken III.bin" file in the games folder.

i play this game on exactly this configuration with windows 7 64bit & 32-bit.
