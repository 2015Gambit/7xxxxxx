CHARM

Charm has eight icons that can be picked up and dropped onto any visible window or application. Each icon performs a different task.

Disable. Disable the selected window.
Reveal. Reveal all hidden windows.
Expand. Expand to show all off-screen windows.
Shutdown. Tell the system that the system is shutting down.
Maximize. Maximize the selected window.
Close. Close the selected window.
Enable. Enable the selected window. 
Hide. Hide the selected window.
