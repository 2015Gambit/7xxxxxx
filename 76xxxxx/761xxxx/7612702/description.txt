This is the program for creating quick notes. Besides the usual note-taking, the program can copy and store the text from other applications - web browsers, text editors, communication programs, such as MSN or ICQ.
Create notes by dragging and dropping selected information on the program window. You can also drag the program and files, to save their location. Pictures, dragged on the window is stored separately. Pictures can be dragged directly from the web pages from web browsers. In this case, the web link of the picture and the image will be stored on your hard drive.

Key features of AgataSoft Drag and Drop Notes:

Quick creation a notes by dragging the text.
Creating notes by dragging links.
Creating notes by dragging the file.
Creating notes and copies of the image by dragging the image.
Creating notes using the "Add" button.
Regulation of transparency of the main window.
