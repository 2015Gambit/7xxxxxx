Paste Manager is a lightweight application that was created in order to provide you with a means of creating hotkeys for pasting various text strings. You can configure a set of personal details, for example, and use the assigned keyboard shortcuts to paste the details in the corresponding field to avoid typing the same information each time. Paste Manager utility to facilitate the entry frequently used text: standard phrases, greetings, blanks letters, e-mail addresses, Internet addresses, logins, passwords. With Paste Manager you can also view the current contents of the clipboard in the dedicated tab of the main window.
