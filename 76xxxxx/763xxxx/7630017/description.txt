These are the installation files for the game "The Lord of the Rings Online", a free-to-play MMORPG by Turbine. This provides an alternative route to download the game for those who don't want to or are unable to use Pando Media Booster (which is the method provided by Turbine). 

This is the client only; in order to play you will need to create an account with Turbine.

Official website: http://www.lotro.com
Turbine's download: http://www.lotro.com/support/download-lotro

Note: This is the high-res version.
