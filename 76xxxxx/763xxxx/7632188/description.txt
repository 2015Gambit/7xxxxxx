disque de reparation windows 7ven ( x86 )




Les options de récupération système peuvent vous aider à réparer Windows en cas d’erreur grave. Pour utiliser ces options, vous avez besoin d’un disque d’installation Windows ou d’accéder aux options de récupération fournies par le fabricant de votre ordinateur.

Ce disque bootable, utilisable sur un système 32bits (x86) vous offre les options suivantes:
- Choix de la langue du clavier à utiliser (il s'agit d'une version internationale)
- Startup Repair (Réparation du démarrage), pour fixer automatiquement les problèmes empêchant windows 7 de démarrer.
- System Restore (Restauration système), pour restaurer votre système à un point de sauvegarde précedent.
- System Image Recovery (Restauration système à partir d'une image), à partir d'une image préalablement crée par sauvegarde.
- Windows Memory Diagnostic (Utilitaire de diagnostic Mémoire), afin de tester physiquement la mémoire.
- Command Prompt (Fenêtre de commande), permettant l'accès à la partition système.

Il s'agit d'un disque multilangue au format iso. Montez le disque dans un lecteur virtuel et gravez.
