Water Drops simulates water drops falling on your wallpaper. You can use the default pool tile background or customize the wallpaper. You can let the drops fall on it automatically, or tap to create new drops.

Features:

    This wallpaper is like the free version, but with adjustable settings:
    Better water reflections
    Quality/battery control
    Automatic drop rate adjustment
    Different drop sizes
    Selectable animation speed
    Zoom in on background
    Improved performance (3x faster)
    Multitouch
    Different reflection styles

Note: If you preferred the classic reflection style, it is now available in the settings menu!

To set the wallpaper, go to:
Home screen -> Menu -> Wallpaper -> Live Wallpapers -> Puddle Drops

What's in this version : (Updated : Sep 12, 2012)
