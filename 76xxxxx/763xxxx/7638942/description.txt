How To Install Game


Mount CoD_CD1, when prompted Install game, if a prompt doesn't come up go to My Computer and double click on
COD1.

During install when prompted to enter key, enter what's in  keysTexT

When prompted to insert disc 2 during install, unmount CoD_CD1 and then mount CoD_CD2 on same drive. If you 
mount the CD's on 2 different drives it will not work.

After CoD_CD2 is finished it will prompt you to insert CD1. Unmount CoD_CD2 and mount CoD_CD1 on same drive. It will
ask you to register, do not register.


Run GAME!!! AND Enjoy!!!
