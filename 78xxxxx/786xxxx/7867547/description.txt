Double Dragon Forever 0.9.1
---------------------------

Double Dragon Forever is a fan made tribute to the first two arcade games featuring the original brawling classic gameplay that made Double Dragon such a hit back in the ┬¡80's.

This is the last version.


A fix to get the game running under Windows Vista/7:

- Right mouse click on "ddragonf.exe".
- Left click on "Properties".
- Click on "Compatibility" tab.
- Check the box "Run this Program in compatibilty mode for:"
- Select "Windows 98 / Windows ME".
- Hit "Ok" at the bottom of the screen.
- Play!
