The Sims 3 patch (OSX) - 1.22.9.011xx 

This torrent containts Patch 1.22 for Mac OS X systems for all regions. Files in individual Region # folders, so grab what you want. 
Starting with High-End Loft Stuff Pack, only 2 regions are released, i.e 
- Region 1 (DVD)
- Region 7 (Digital)
So, those whose base game is on Region 2 or 3 or 5 can still use the HELS' (and other games) patch from the Region 1 folder.
This is also the last version where the individual EPs/SPs are patched. After this, only the base game gets patched and all the EP/SP patched files are routed to the base game folders.

For more info on how to locate or understand the game versions and to get current patches, read the FAQs on MTS' SimsWiki :
-   http://simswiki.info/wiki.php?title=Game_Help:TS3_Patching_Find_Gameversion
-   http://simswiki.info/wiki.php?title=Game_Help:TS3_Patching_Versions#Understanding_Region_codes
-   http://simswiki.info/wiki.php?title=Game_Help:TS3_Patching/Patch_Download_OSX

These are untouched, as you can see from the MD5 checksums that you can compare on other sites or from these links. The base game's link is of course superceded by the current patch version numbers as EA do not keep archived copies of MAC OSX patches. You just have to take my word for it.
