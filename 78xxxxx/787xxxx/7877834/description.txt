Secure your Java applications using Java EE built-in features as well as the well-known 
Spring Security framework. Utilize related recipes for testing various Java EE 
technologies including JPA, EJB, JSF, and Web services. Explore various ways to 
extend a Java EE environment with the use of additional dynamic languages as well 
as frameworks. A learn-by-example based approach that focuses on key concepts to 
provide the foundation to solve real world problems.
