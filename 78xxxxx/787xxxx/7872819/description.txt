Instructions to install:
    METHOD #1 (WinRAR)
        1) Open the .mdf file with WinRAR
        2) Extract contents to a folder
        3) Run the autorun executable
    METHOD #2 (Mounting programs)
        1) Rename the extension of the .mdf file to .iso
        2) Mount with a mounting program (PowerISO, etc)

You will also need the Japanese language pack and made it as your secondary language of your computer to be able to read the text. Google for how-tos.
