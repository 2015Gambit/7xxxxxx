Ripple Lock v3.6

Requirements: Android version 2.1 and higher

Is still envy of the brilliant lock screen of Android 3.0? Now, as long as your phone is Android system, you will be able to experience the beauty of the lock screen.


Only use one button, you can choose to use this ripple lock screen or system default screen.
Ripple Lock has plenty of setting options: font color, font size, background, and so on. 
Drag the inside circle to the outside circle can unlock screen.
This program will continuously update and add more colorful lock screen. Please suggestions to me at any time.

What's in this version:
1. Fix double lock issue for some phones
2. Fix the issue when app been killed by some process management application
3. Remove the "Fix double lock issue" and "Use system security lock" options

This app has no advertisements
