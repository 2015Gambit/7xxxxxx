Alliance of Valiant Arms (A.V.A) - a First-Person Online Shooter

This here is the SFX.exe which when doubleclicked extracts the game in the same directory the SFX.exe is in.

Aeria Ignite does not install with this and the game can be launcher from NWZLauncher.exe, which is in the main directory where the game was extracted.

An Aeria account is required to play the game.
