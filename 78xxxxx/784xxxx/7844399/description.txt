PhotoFiltre Studio

PhotoFiltre Studio is a complete image retouching program. It allows you to do simple or advanced
adjustments to an image and apply a vast range of filters on it. It is simple and intuitive to use, and 
has an easy learning curve. The toolbar, giving you access to the standard filters with just a few 
clicks, gives PhotoFiltre Studio a robust look. PhotoFiltre Studio also has layer manager (with Alpha
channel), advanced brushes, nozzles (or tubes), red eye corrector, batch module and lot of other 
powerful tools.

Supported OS:

Vista, Windows 7, Win XP, Win NT, Win 2000

Homepage: http://www.photofiltre-studio.com/download-en.htm

Installation Instructions:

1. Install program.
2. Use given key to register application.
3. Done, Enjoy.
