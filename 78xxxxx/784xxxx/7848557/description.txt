If you want to get into developing web sites, the most important thing you'll need
 

is a solid understanding of Hypertext Markup Language, or HTMLthe most 

common language used to write web site content.

 The most recent version of the language is HTML5, and it contains a whole host

 of new features to give you more power when creating websites
