Title: The Devil Inside
Genre: action / horror/ 3rd-person
Released: 31st October 2000
Developer: GameSquad
Publisher: Cryo Interactive
Description:
The Devil Inside is an action/adventure survival horror game set inside a Hollywood mansion, Shadow Gate, that has been infested by an executed mass murderer and his undead legion. You play as Dave Ackland, ex-cop turned television star, and it's your job to send the murderous souls and their leader back to the depths of hell. The twist is that you are doing this on live television in front of a cheering studio audience. The live television setting combined with the eerie atmosphere make for a suspenseful action thriller that is sure to keep players glued to their screens!

How to play:
1) Write DEVILINSIDE.cue on a disc or mount it on a virtual drive;
2) Run Setup.exe and install the game (it will start automatically if you have autorun enabled);
3) Extract DIPatchUS.exe somewhere on your hard drive, then run DIPatcher.exe and patch the game;
4) Play the game with the disc inserted.

! Notice
Tested and working fine on Windows 7 x64 with all the latest updates installed.

! Notice
The disc must be inserted in the first CD/DVD-ROM drive for the game to work.
