English

Description: AVG Internet Security is a protection tool that aims to make the experience the user has with the Internet as secure as possible.

All with automatic settings that dispense waste of time, which is possible with the setting details for a performance compatible with the machine.

Besides working on blocking and removing common viruses, this software manages online activities and bar the entry of any suspicious file. A powerful firewall ensures the lock of any external connection disallowed, leaving the responsibility of the user to define who can and can not access the data on the computer.


Português (Brasil)

Descrição: O AVG Internet Security é uma ferramenta de proteção que tem como objetivo deixar a experiência que o usuário tem com a internet o mais segura possível.

Tudo isso com configurações automáticas que dispensam perda de tempo, o que é possível com o ajuste de detalhes para obter um desempenho compatível com a máquina.

Além de trabalhar no bloqueio e remoção de vírus comuns, este software gerencia as atividades online e barra a entrada de qualquer arquivo suspeito. Um firewall poderoso garante o bloqueio de qualquer conexão externa não permitida, deixando a cargo do usuário definir quem pode ou não ter acesso aos dados presentes no computador.
