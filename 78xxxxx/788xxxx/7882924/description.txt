Introduction

Welcome to Teach Yourself UNIX in 24 Hours! This book has been designed so it is helpful
for both begining users and those with previous UNIX experience. This text is helpful as
a guide, as well as a tutorial. The reader of this book is assumed to be intelligent, but
no familiarity with UNIX is expected.
