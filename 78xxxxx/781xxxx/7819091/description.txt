This is LuaJIT 2.0.0 (final release) binaries for Windows.
LuaJIT git HEAD (last change on November 8, 2012) was compiled 
using MS Visual Studio 2005 with /DLUAJIT_ENABLE_LUA52COMPAT
option (see http://luajit.org/extensions.html#lua52).

Latest version of LuaJIT binaries for Windows is available at 
http://luajit4win.no-ip.org
