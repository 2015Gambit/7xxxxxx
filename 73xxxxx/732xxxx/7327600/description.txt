This software offers a solution for users who want to retrieve file-level passwords for one or more Word files. The user can specify a list of files or an entire folder to be processed. There is an option to recover using brute force or a user specified dictionary. The brute force option has a selection of options to increase speed of recovery, such as: using a user specified minimum, maximum or exact length of password or using a limited character set, number set or a custom character set. Word 2000 or higher required.
