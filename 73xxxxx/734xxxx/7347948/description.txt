Application: Minecraft Multi-Version Installer

Description: 
Java Minecraft installer that includes all versions of Minecraft since Indev 2.23.10. (102 different versions)

Platform Supported: Windows
Java Version: 1.6+
Minecraft Versions: Indev 2.23.10 - Snapshot 12w23b

Minecraft Snapshot 12w23b

Instructions:
1. Run "Minecraft Multi-Version Installer"
2. Select version of Minecaft you would like to install from combobox
3. Click "install Game"
4. Click "Install Launcher" (Places shortcut on desktop)
5. Play!
