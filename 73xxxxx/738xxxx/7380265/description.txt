This is my collection of the music from Avatar: The Last Air Bender. You can find the music on youtube but I thought it'd be nice to have them all collected with a nice album picture. Each sound file is 128kbps stereo and it's been sorted into what I think is chronological, by looking up each episode that contains the song.
