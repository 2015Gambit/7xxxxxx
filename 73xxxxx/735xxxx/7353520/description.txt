Description:
Process Lasso e application that allows you to manually or automatically controlling running processes in the system, which in turn leads to increased performance and stability when working with computers. Gives you complete control over the process and gives you full information about them. The program monitors the processes that consume most of CPU power and automatically limited them. You can determine which processes should be excluded automatically when you start. Process Lasso is able to determine the priorities of processes certain applications whenever they start.

Installation:
1. Install Process Lasso
2. Exit/terminate/shutdown ProcessLasso.exe and ProcessGovernor.exe (make sure both exe file not exist in taskmanager)
3. Copy patch to target folder
4. Apply patch
5. Done
