Here's a RPG Maker XP with a Keygen.

--------------Installation--------------

Double click Setup Menu and Install Both Options. 

Then Open Program and click already Purchased,

Generate Key

Then Type in the Keygens Serial Code, 

Click Generate by Url

Type the URL into the Keygen and then click Generate 

Type the Code into the first Box 

Enjoy Creation!!
