Description:
PhotoFiltre Studio - an enhanced version of the program for retouching graphic images, has a number of additional features. The program is easy to use and has all the basic editing tools such as brightness, contrast, color saturation and gamma correction features. In addition, the disposal of the user is more than a hundred filters for image processing, such as watercolor, pastel, the effect of breaking up on puzzles, etc.

Those filters that are used most often made on the toolbar for easy access. Among other features of PhotoFiltre Studio can note the use of layers, advanced selection and save the selection as a separate file for later use.
