This is an archive snapshot of the TasteyServ Minecraft map Circa June 2012.
This is a huge multiplayer map with bukkit included in the archive. Rail lines extend throughout the map.
Many Cities can be found around the map as well.
In this snapshot is everything we had on the server when it was archived, including the fully rendered dynmap, and all the plugins.

Size of archive (gzip): 2.8 GB
Size of Map after extraction: 3.8 GB

Enjoy!
