1. Install Dreamweaver As a Trial by clicking "Try, I would like to try adobe dreamweaver cs6 for a limited time"
2. Before clicking Accept on the License Agreement choose your language on the upper left hand corner then click Accept
3. Now Install and wait for installation to be complete
4.Once thats done Open up Dreamweaver.exe and activate your 30 free trial then close Dreamweaver once you have done that
5.Inside the Patch Folder there will be a amtlib.dll file compatible for both 32bit and 64bit computers
6. Copy and paste the amtlib.dll from the patch folder over to your C:Program Files AdobeAdobe Dreamweaver CS6 or C:Program Files (x86)AdobeAdobe Dreamweaver CS6 Directory
7. Now run Dreamweaver CS6 you should now never recieve any pop ups referring to you have so and so days for your trial
