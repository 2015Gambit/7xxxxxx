This application retrives and extracts e-mail addresses reciently posted on Twitter.
 
It will query twitter at fixed intervals searching for recently posted email addresses.
 
You can harvest thousands of emails a day.

It also has a function to extract email addresses from inputted url's.
 
As well as email extraction from a text file.



To operate application run the executable.

Select where you want the file to be stored.
Sselect your refresh rate (5 - 10 works best)

click Fly!.



To stop application click on Land!.

You can count your emails in your list by using the Count Emails button.

You can remove duplicate emails using the Remove duplicates button.



Have Fun ! :D
