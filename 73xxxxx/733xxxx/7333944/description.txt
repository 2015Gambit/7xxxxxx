The Offical Blaze GUI (In Game Menu)



Offers a list of up to date non steam servers right from the Counter-Strike: Source game menu, easy to use just click and connect.




Screenshot here:
 
http://pictat.com/i/2012/6/6/28554screenshot.jpg

Blaze Clan Website here:
 
http://blaze-clan.co.uk/




Instructions:

Go to the directory where your game is saved.

Something like this:

C:Program Files (x86)Steamsteamapps(Your Account)counter-strike source.


Under this directory you should see a folder named "cstrike".


Copy Paste the cstrike from the download to this directory.



Have Fun!
