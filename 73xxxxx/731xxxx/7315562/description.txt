Fallout:New Vegas
=====================================================================================
Contains:
----------
-A Iso File for FNV
-Update 7 Installer
-SKIDROW latest Installation(No Disc Mod)
----------

Installation:
--------------
1. Extract .Torrent With a Torrent Extracter
   -www.bitlord.com-
2. Mount Disc using Power Iso.
   -www.Poweriso.com-
3. Navigate to Mounted Drive and click Setup.exe and install FNV.
4. Open Update Mod folder and install the update 7, then navigate to the
   Bethesda Softworks folder click Fallout New Vegas folder, Now back to the Update Mod folder click the SKIDROW folder and drag all the files to the Fallout New Vegas folder 'Move and Replace'.


Enjoy : )
